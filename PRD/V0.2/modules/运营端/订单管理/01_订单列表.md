## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 运营端订单列表 |
| 所属端 | 运营端 |
| 入口路径 | 订单管理 > 门店订单 / 分红订单 / 平台订单 / 全部订单 |
| 使用角色 | 平台管理员、审核客服、订单客服、财务、售后、风控 |

## 2. 订单范围与页面拆分

| 订单类型 | 是否显示 | 说明 |
|---|---|---|
| 门店订单 | 显示 | 平台可查看与抽佣，但审核由门店自审 |
| 分红订单 | 显示 | 平台审核、分配资方、处理合同/支付/授权/分账 |
| 平台订单 | 显示 | 原低费率订单改名为平台订单，由平台匹配资方并主控审核财务 |

订单管理不再只做一个大杂烩列表。三类订单的审核主体、资金流、操作按钮、展示重点不同，运营端订单管理拆成四个入口：

```
订单管理
├─ 门店订单
├─ 分红订单
├─ 平台订单
└─ 全部订单
```

| 页面 | 定位 | 日常使用方式 |
|---|---|---|
| 门店订单 | 门店自营订单监管与抽佣查看 | 平台主要看商户、发货、客诉、逾期、抽佣和钱包入账，不参与常规审核 |
| 分红订单 | 门店 + 资方共同参与的订单工作台 | 平台重点处理资方分配、审核、合同、风控、分账 |
| 平台订单 | 平台/资方主控订单工作台 | 平台重点处理资方、审核、合同、渠道佣金、发货主体和资金进度 |
| 全部订单 | 跨类型全局搜索与汇总 | 只作为查询和汇总入口，不作为日常主工作台 |

默认不进入「全部订单」。运营端可根据角色配置默认入口，例如审核客服默认进入「分红订单」或「平台订单」，租后客服默认进入逾期状态视图。

### 2.1 三类列表展示重点

| 列表 | 展示重点 | 操作重点 |
|---|---|---|
| 门店订单 | 商户/门店、办单员工、客户资料、商品/租期/金额、商家审核状态、发货状态、平台抽佣、商家钱包入账、逾期/客诉 | 查看、备注、抽佣查看、客诉处理、租后查看 |
| 分红订单 | 商户/门店、设备价、门店出资比例、资方出资比例、资方分配、风控、合同、每期分账、门店钱包/资方钱包、平台抽佣 | 分配资方、审核、改套餐、发起合同、查看分账、联系客服、要求补资料 |
| 平台订单 | 商户/门店、资方、风控、合同、平台/资方资金、渠道来源、渠道佣金、发货主体、回款/代扣 | 分配资方、审核、改套餐、发起合同、查看资方收益、查看渠道佣金、切换发货主体 |

## 4. 筛选区

| 字段 | 类型 | 说明 |
|---|---|---|
| 订单号 | 文本 | 支持精确查询 |
| 商品名称 | 文本 | 支持模糊查询 |
| 用户姓名 | 文本 | 查询时后台按权限处理明文匹配，前端列表脱敏展示 |
| 手机号 | 文本 | 支持手机号后四位或完整手机号查询 |
| 商户/门店 | 下拉/搜索 | 可筛选不同商家订单 |
| 订单类型 | 下拉 | 门店订单 / 分红订单 / 平台订单 |
| 订单状态 | 下拉 | 与状态标签一致 |
| 下单渠道 | 下拉 | 支付宝小程序、微信小程序、门店扫码、渠道推广等 |
| 支付方式 | 下拉 | 微信支付、支付宝、租赁行业代扣、银联代扣等 |
| 时间范围 | 日期区间 | 下单时间、审核时间、到期时间可切换 |

## 5. 状态标签

| 标签 | 说明 |
|---|---|
| 全部 | 当前权限下可见订单总数 |
| 待付款 | 客户已提交但未完成首笔支付/冻结 |
| 待审核 | 需要平台或门店审核 |
| 待分配资方 | 分红订单、平台订单需要选择资方 |
| 待签约 | 合同、授权、公证等未完成 |
| 待支付 | 客户待支付首期、服务费、公证费等 |
| 待发货 | 审核/签约/支付完成，等待门店或平台发货 |
| 待收货 | 已发货未确认收货 |
| 待归还 | 租期中或即将归还 |
| 已逾期 | 已超过应还/应付时间 |
| 归还中 | 已发起归还流程 |
| 已完成 | 订单正常完结 |
| 已购买 | 用户完成留购/买断 |
| 已关闭 | 取消、审核拒绝、退款关闭等 |

## 7. 操作按钮

| 按钮 | 显示条件 | 点击后 | 日志要求 |
|---|---|---|---|
| 订单详情 | 有查看权限 | 打开详情页或右侧抽屉 | 记录查看敏感信息 |
| 风控信息 | 有风控权限 | 打开风控详情 | 记录查看 |
| 审核资料 | 有审核权限 | 打开审核资料抽屉 | 记录查看 |
| 分配人员 | 待分配员工 | 打开人员选择弹窗 | 记录分配人 |
| 订单分配 | 可转单 | 打开商户/客服分配弹窗 | 记录转单前后主体 |
| 分配资方 | 分红/平台订单且待分配资方 | 打开资方选择弹窗 | 记录资方与比例 |
| 联系客服 | 商家发起的平台订单 | 调起 IM 并推送订单摘要 | 记录会话关联 |
| 取消订单 | 可取消状态 | 二次确认后取消 | 必须记录原因 |
| 审核通过 | 待审核 | 二次确认后进入下一状态 | 必须记录审核意见 |
| 审核不通过 | 待审核 | 填原因后拒绝 | 必须记录原因 |
| 加黑名单 | 有风控权限 | 二次确认后加入黑名单 | 必须记录原因 |
| 交易快照 | 有查看权限 | 打开订单快照 | 记录查看 |
| 发起征信/风控授权签署 | 待授权 | 调用签署接口 | 记录接口返回 |
| 发起流水报告 | 待补充风控 | 调用报告接口 | 记录接口返回 |
| 发起补充合同 | 待补充合同 | 调用 e 签宝 | 记录合同编号与返回 |

## 8. 关键交互

```
[查询] ──> 按筛选条件刷新订单卡片
[展开] ──> 显示更多筛选项
[状态标签] ──> 切换订单状态并刷新数量
[订单详情] ──> 打开详情页/抽屉，查看完整资料
[审核资料] ──> 打开资料抽屉，支持身份证、营业执照、授权文件等预览
[分配资方] ──> 选择资方 ──> 生成资方占比/分账规则
[联系客服] ──> 调起 IM ──> 自动推送订单摘要给客服
[审核通过] ──> 二次确认 ──> 进入签约/支付/发货下一步
[审核不通过] ──> 填拒绝原因 ──> 订单关闭或退回
```

## 10. 敏感信息规则

1. 列表默认脱敏展示姓名、手机号、身份证号、地址、IP。
2. 查看完整信息需要权限，并写入查看日志。
3. 导出订单需要单独权限，导出文件记录操作人、时间、筛选条件。
4. 文档、测试数据、演示环境不得使用真实生产订单样例。
