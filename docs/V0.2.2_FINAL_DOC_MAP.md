## 四、当前已确认的关键口径

| 主题 | 当前口径 |
|---|---|
| 一期范围 | 安心用长租 + 体验租短租 V1 + IM 客服基础能力 |
| 三类订单展示 | 商家订单 / 联营订单 / 平台订单 |
| 申请购买价 | 一期按 A 口径:剩余应付租金 + 保证金;折旧余值口径后期再评估 |
| 监管锁 | 适用于苹果类手机、平板、手表等配置品类;其它品类按签收和交付证据结算 |
| 短租责任 | 非车辆质量问题导致的事故、违章、损坏、丢车原则上由承租人承担 |
| 财务 | 四账分离:订单业务账、支付流水账、钱包/商家账户账、财务总账/分录账 |
| C 端隔离 | 不展示内部资金来源、合作模式、风控结论、服务费拆分、商家结算价、平台利润、返点规则等内部信息 |
