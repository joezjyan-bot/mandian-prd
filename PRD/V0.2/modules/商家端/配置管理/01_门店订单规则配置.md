# 门店订单规则配置

> 页面级 PRD 草案。
> 目标：让商家在自己的权限范围内配置门店订单的租赁规则、费率、增值服务、合同、公证、发货和审核策略，同时不影响分红订单和平台订单的运营端主控规则。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 门店订单规则配置 |
| 所属端 | 商家 PC 端 |
| 入口路径 | 商家中心 > 配置管理 > 门店订单规则 |
| 使用角色 | 商家老板、商家管理员 |
| 核心目标 | 配置商家自营门店订单的办单、审核、费率、增值服务、合同、发货、归还和员工权限 |

该页面只管门店订单。分红订单、平台订单的审核、资方、合同、风控、分账主规则由运营端配置。

---

## 2. 核心口径

1. 门店订单是商家自营订单，默认由商家/门店自行审核。
2. 商家可配置门店订单可用商品、租期、增值服务、发货和归还规则。
3. 平台保留抽佣、监管、客诉、租后、日志和高风险介入能力。
4. 平台抽佣默认 2%，商家端只展示已生效规则，不允许普通商家自行修改平台抽佣。
5. 商家配置不得覆盖运营端全局安全底线，例如敏感数据脱敏、黑名单提示、操作日志。
6. 员工账号只可按授权使用配置，不允许修改核心配置。
7. 规则变更必须保存版本，已生成订单和二维码按生成时快照执行。

---

## 3. 页面结构

```text
门店订单规则配置
├─ 基础规则
├─ 租赁模式与租期

├─ 增值服务
```

## 4. 基础规则

| 字段 | 类型 | 说明 |
|---|---|---|
| 是否启用门店订单 | 开关 | 关闭后门店订单办单助手不展示 |
| 默认审核主体 | 只读/配置 | 商家/门店自审 |
| 是否需要平台复核 | 开关 | 默认否，特殊商家可由运营端强制 |
| 订单自动关闭时间 | 数字 | 客户未支付/未提交资料超时关闭 |
| 是否允许改价 | 开关 | 老板/管理员可改，员工默认不可改 |
| 是否允许线下收款 | 开关 | 建议仅老板/管理员可用 |
| 是否启用黑名单提示 | 只读/开关 | 可由平台强制启用 |
| 平台抽佣比例 | 只读 | 默认 2%，来自运营端配置 |

---

## 5. 租赁模式与租期

商家从运营端启用的租赁模式中选择门店订单可用范围。

| 字段 | 类型 | 说明 |
|---|---|---|
| 可用租赁模式 | 多选 | 长租、短租 |
| 可用计费单位 | 多选 | 小时、天、周、月 |
| 可用租期 | 配置 | 例如 2 小时、1 天、1 周、3/6/9/12 月 |
| 起租规则 | 下拉 | 支付后、发货后、客户确认收货后、门店交付后 |
| 到期规则 | 多选 | 归还、续租、留购 |
| 超时计费 | 配置 | 短租必须配置或继承平台默认 |
| 是否要求绑定设备 | 开关 | 短租默认必须 |

规则：

1. 商家不能启用运营端已停用的租赁模式。
2. 商品规格只能选择当前配置允许的租期。
3. 短租无设备库存时，不能生成可下单二维码。
4. 修改租期后，历史订单和旧二维码继续按旧快照。

---

## 6. 费率与押金

| 字段 | 类型 | 说明 |
|---|---|---|
| 首期规则 | 配置 | 固定金额、比例、按计算器结果 |
| 后期账单规则 | 配置 | 按期、按天、按小时 |
| 押金规则 | 配置 | 固定、比例、免押、补充押金 |
| 留购规则 | 配置 | 到期留购、提前买断、不可留购 |
| 服务费规则 | 关联 | 从增值服务读取 |
| 逾期费用规则 | 配置 | 可选，需展示给客户 |
| 允许员工改价 | 开关 | 默认关闭 |
| 改价审批 | 开关 | 超过阈值需老板审批 |

门店订单可由商家自定义经营规则，但客户扫码页面必须清晰展示账单、押金、服务费、公证费、留购价。

---

## 7. 增值服务

| 字段 | 类型 | 说明 |
|---|---|---|
| 可用增值服务 | 多选 | 商家自建服务和平台允许服务 |
| 是否必选 | 开关 | 必选服务客户不可取消 |
| 是否计入首期 | 开关 | 影响首期实付 |
| 是否计入账单 | 开关 | 影响后期账单 |
| 是否可退 | 开关 | 影响退款 |
| 是否允许改价 | 开关 | 商家审核或改套餐时使用 |
| 财务归属 | 下拉 | 商家收入、押金、平台代收、成本 |

规则：

1. 商家端增值服务默认只用于门店订单。
2. 平台可限制服务类型和字段范围。
3. 商品绑定服务后，办单助手、订单详情、客户账单必须拆开展示。

---

## 8. 合同与授权

| 字段 | 类型 | 说明 |
|---|---|---|
| 是否启用合同 | 开关 | 商家自营订单是否需要合同 |
| 合同模板 | 选择 | 商家模板或平台提供模板 |
| 签署服务 | 下拉 | e 签宝或平台配置服务 |
| 是否自动发起 | 开关 | 审核通过后自动发起 |
| 是否启用补充合同 | 开关 | 改价、续租、留购时可用 |
| 商家签章授权状态 | 只读 | 来自入驻认证 |
| 是否启用公证 | 开关 | 可按商品或订单配置 |
| 公证费用承担方 | 下拉 | 客户、商家、平台 |

如果商家启用平台提供的合同/公证能力，可能触发商家接口计费。

---

## 9. 发货与归还

| 字段 | 类型 | 说明 |
|---|---|---|
| 默认发货主体 | 下拉 | 门店、商家 |
| 发货方式 | 多选 | 当面交付、门店配送、快递发货 |
| 是否要求交付材料 | 开关 | 设备照、人机/人车合照 |
| 是否要求客户确认收货 | 开关 | 影响起租规则 |
| 归还方式 | 多选 | 到店、快递、上门 |
| 归还验收角色 | 下拉 | 门店员工、店长、管理员 |
| 验收异常处理 | 配置 | 赔付、争议、维修、入库 |
| 短租设备入库 | 开关 | 验收通过后回到可租 |

默认门店发货，但商家可以按自己的门店配置发货和归还方式。

---

## 10. 员工办单权限

| 字段 | 类型 | 说明 |
|---|---|---|
| 店员可用门店订单办单助手 | 开关 | 默认可配置 |
| 店员是否可改价 | 开关 | 默认否 |
| 店员是否可审核门店订单 | 开关 | 默认否 |
| 店员是否可上传交付材料 | 开关 | 默认是 |
| 店员是否可归还验收 | 开关 | 可配置 |
| 店员是否可查看客户资料 | 下拉 | 脱敏、部分明文、不可见 |

员工权限最终仍由员工管理里的角色和门店授权共同决定。

---

## 11. 发布与版本

| 动作 | 规则 |
|---|---|
| 保存草稿 | 不影响线上 |
| 发布配置 | 校验规则完整性，生成版本 |
| 生效时间 | 支持立即或指定时间 |
| 回滚 | 生成新版本，保留日志 |
| 影响预览 | 展示影响商品、规格、办单助手和未下单二维码 |

配置发布必须写入操作日志，记录操作人、变更前后值、影响范围。
