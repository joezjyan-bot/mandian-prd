## 守的红线(已在原型中落实)

1. **C 端红线**:C 端各页完全不出现合作模式(商家/联营/平台)、资方/资金来源、分账、平台服务费拆分、商家结算价、渠道佣金、风控字眼。客户只看到"租门店一台设备 + 每期应付 + 保证金 + 到期可还/可续/可买"。
2. **门店端内部端**:门店端是给店员/老板用的,可见合作模式(商家/联营/平台订单),但联营配资比例、分账/收益预览标注"仅内部可见、不给客户";二维码不展示成本价/内部资金规则。
3. **合同三方制**:签约线框中合同三方仅为门店(甲)、客户(乙)、平台(丙),不出现资方,避免融资租赁定性。
4. **到期三选一同等展示**:归还 / 继续使用 / 申请购买三个选项同一视觉权重,明确标注不默认勾选、无"到期自动购买 / 自动转移所有权"。
5. **人脸隐私红线**:门店端交付/归还的"人机合照"标注"仅存比对结果",不建立可检索人脸库。
6. **权限边界**:商家/门店店员不可见平台利润 / 服务费拆分 / 钱包 / 分账 / 其他商家;不可编辑商品、不可见成本价;后端按角色校验不只靠前端隐藏。
7. **必须本人**:实名人脸、收货确认、合同签署、首期支付、代扣绑卡均须客户本人,不可门店/他人代替。
8. **隐私脱敏**:身份证号、银行卡号、手机号、客户姓名等均脱敏展示;平台不留存完整卡号。
9. **损坏赔付**:归还验收判定损坏/缺件走人工复核,不在门店端直接扣款。
