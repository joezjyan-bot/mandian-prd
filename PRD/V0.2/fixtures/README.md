# V0.2 测试数据样例

> 用途：给开发、测试、Claude 和后续自动化用例使用。

---

## 文件说明

| 文件 | 用途 |
|---|---|
| `seed-data.demo.json` | 三类订单、商家门店、角色账号、商品规格、设备、账单、分账和钱包种子数据 |
| `mock-callbacks.demo.json` | 支付、合同、公证、物流、投诉等 Mock 回调样例 |

---

## 使用约束

1. 所有数据都是占位数据，只能用于开发和测试。
2. 所有数据必须带 `tenant_demo_main` 或 `is_mock = true`。
3. 不允许把真实客户资料、真实订单、真实通道配置复制进样例。
4. 自动化测试可按 `tenant_id` 和 `is_mock` 清理数据。
