# 增值服务配置

> **Stage 6 术语同步(2026-05-27)**: 本文档已按 Stage 6 统一为商家、联营、平台订单、订单结算款、我的钱包、履约中、逾期费用、留购、保证金等展示术语；数据库字段、API 路径、英文枚举保持不变。

> 页面级 PRD 草案。
> 目标：统一配置服务费、公证费、设备管理费、保险、上门服务等可叠加费用，并同步到商品、办单助手、订单改套餐和财务分账。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 增值服务配置 |
| 所属端 | 运营端，商家 PC 端可维护商家订单增值服务 |
| 入口路径 | 配置管理 > 增值服务配置 |
| 使用角色 | 平台管理员、商品运营、财务、商家老板、商家管理员 |
| 核心目标 | 建立增值服务库，按订单类型、商品、规格、租赁模式、办单助手入口绑定，并进入订单费用和财务流水 |

增值服务不能只写在商品备注里。它会影响客户费用展示、办单助手计算、客服改套餐、订单账单、分账、退款和对账。

---

## 2. 核心口径

1. 运营端配置的增值服务默认用于联营订单、平台订单。
2. 商家端配置的增值服务默认用于商家订单。
3. 商品新增/编辑时可以选择可用增值服务，并配置是否必选、是否可改价。
4. 办单助手按入口、商品、规格、租赁模式展示可选增值服务。
5. 客服审核改套餐时，可以按权限添加、移除或改价增值服务。
6. 每个订单保存增值服务快照，后续配置变更不影响历史订单。
7. 增值服务是否参与分账、退款、首期收取、账单摊销必须配置清楚。

---

## 3. 增值服务类型

| 类型 | 示例 | 说明 |
|---|---|---|
| 固定服务费 | 平台服务费、设备管理费 | 固定金额收取 |
| 公证类费用 | 公证费、电子证据服务费 | 可和公证链路联动 |
| 风控类费用 | 大数据查询费、征信授权费 | 可和接口计费联动 |
| 保险/保障 | 意外险、碎屏保障、车辆保障 | 可选或必选 |
| 履约服务 | 上门安装、同城配送、取还服务 | 可按短租或区域配置 |
| 保证金补充 | 补充保证金、风险保证金 | 可作为保证金类，不一定参与收入 |
| 其他 | 自定义服务 | 需配置财务归属 |

## 6. 商品绑定

| 绑定层级 | 说明 |
|---|---|
| 类目 | 适用于某类商品，例如电动车都收设备管理费 |
| 商品 | 适用于指定商品 |
| 规格 | 适用于指定规格，例如二手规格启用保障服务 |
| 办单助手入口 | 商家订单、联营订单、平台订单分别配置 |
| 商家覆盖 | 平台商品复制给商家后，商家可按权限增减商家订单服务 |

绑定规则：

1. 商品编辑页从增值服务库选择，不重复创建服务。
2. 同一服务可在不同入口配置不同金额。
3. 联营订单和平台订单的增值服务由运营端主控。
4. 商家订单的增值服务由商家端主控，但平台可设置可用范围和监管。

---

## 7. 办单助手展示

| 入口 | 增值服务来源 | 规则 |
|---|---|---|
| 商家订单 | 商家配置 | 店员按权限选择，必选项不可取消 |
| 联营订单 | 运营配置 | 进入平台审核和分账 |
| 平台订单 | 运营配置 | 进入平台审核、资方分账或平台收入 |

办单助手费用预览必须把增值服务拆开展示，不能只合并成总价。

---

## 8. 客服改套餐

订单审核或详情页改套餐时，客服可以看到：

| 字段 | 说明 |
|---|---|
| 原增值服务 | 下单时快照 |
| 可添加服务 | 当前商品和订单类型允许的服务 |
| 可移除服务 | 非必选服务 |
| 可改价服务 | 配置允许改价的服务 |
| 费用影响 | 首期、后期账单、总租金、留购价、分账预览 |
| 客户确认 | 改价后是否需要客户重新确认 |

改套餐必须生成价格方案新版本，保留旧版本和操作原因。

---

## 9. 财务与分账

| 配置 | 财务影响 |
|---|---|
| 平台收入 | 入平台收入账户 |
| 商家收入 | 入商家/门店钱包 |
| 资方收入 | 入资方账户 |
| 渠道佣金 | 入渠道账户待结算 |
| 保证金类 | 进入保证金账户或冻结余额 |
| 不参与分账 | 只作为订单展示项或成本项 |

退款时必须按服务配置判断是否可退、退给谁、是否冲减已分账。
