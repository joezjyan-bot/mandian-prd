# 信审管理

- 左侧菜单：`信审管理`
- 子菜单：`信审管理`

## UI 结构

```text
信审管理
├─ 查询区
│  └─ 信审人员
├─ 操作区
│  ├─ 查询
│  └─ 添加

   ├─ 信审名称
```

## 查询区字段

| 字段 | 类型 | 占位 | 说明 |
|---|---|---|---|
| 信审人员 | 输入框 | `请输入` | 按信审人员名称或关键词查询 |

## 按钮与点击反馈

| 点击位置 | 结果 | 风险 |
|---|---|---|
| `查 询` | 刷新表格；当前为空数据，展示 `暂无数据` | 低 |
| `添 加` | 打开添加信审人员弹窗 | 中，确定会新增或绑定信审人员 |

## 添加弹窗

```text
点击 添加
└─ 弹窗：请确认
   ├─ * 成员名称：下拉/组合框，placeholder `请选择`
   ├─ * 手机号
   ├─ 取消
   └─ 确定
```

## 空状态

- 当前列表为空，表格区域展示 `暂无数据`。
- 添加弹窗可取消返回列表。

## 权限与审计

- 信审人员拥有订单审核相关权限，应与 `权限管理 / 成员管理` 的账号体系打通。
- 新增、停用、移除信审人员必须记录操作人和时间。
