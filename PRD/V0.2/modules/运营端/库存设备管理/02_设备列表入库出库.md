# 设备列表、入库、出库与归还验收

> **Stage 6 术语同步(2026-05-27)**: 本文档已按 Stage 6 统一为商家、联营、平台订单、订单结算款、我的钱包、履约中、逾期费用、留购、保证金等展示术语；数据库字段、API 路径、英文枚举保持不变。

> 页面级 PRD 草案。
> 目标：把设备库存模型落成可操作页面，主要支撑短租设备的入库、锁定、出库、归还、验收和维修。长租只在发货/交付节点填写设备识别码，不走库存状态。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 设备列表、入库、出库与归还验收 |
| 所属端 | 运营端，商家 PC 端和门店手机端按权限复用部分能力 |
| 入口路径 | 库存设备管理 > 设备列表 / 入库 / 出库 / 归还验收 |
| 使用角色 | 平台设备运营、商家老板、门店店长、门店员工、仓库人员、售后 |
| 核心目标 | 管理短租真实设备生命周期，确保短租订单可追踪、可验收、可复租 |

---

## 2. 核心口径

1. 商品不是库存，商品用于展示和价格配置；设备才是真实可出租资产。
2. 短租订单必须锁定具体设备，不能只扣商品库存数量。
3. 长租不进入设备库存状态；长租发货/交付时在发货表单填写 IMEI/SN/VIN 等设备识别码。
4. 设备入库、锁定、出库、出租、归还、维修、下架都必须保留操作日志。
5. 默认门店发货场景下，门店员工可处理自己门店的交付和归还验收。
6. 平台订单或联营订单如配置平台/资方发货，门店端只读或协助。
7. 短租设备状态必须和订单状态、监管锁接口状态、交付照片、财务赔付联动。
8. 已出租短租设备不能重复出库，已锁定短租设备不能被其他二维码或订单再次锁定。

## 4. 新增设备与入库

### 4.1 单台新增

| 字段 | 类型 | 说明 |
|---|---|---|
| 设备码 | 文本/生成 | 必填，系统唯一 |
| 商品 | 选择 | 必填 |
| 规格 | 选择 | 必填 |
| 所属主体 | 选择 | 平台、商家、门店 |
| 入库仓库 | 选择 | 必填 |
| SN/IMEI/VIN | 文本 | 按类目必填 |
| 设备成色 | 下拉 | 全新、99新、95新、二手等 |
| 入库照片 | 图片 | 可配置必填 |
| 备注 | 文本 | 可选 |

### 4.2 批量导入

| 功能 | 说明 |
|---|---|
| 下载模板 | 商品、规格、设备码、SN/IMEI/VIN、仓库 |
| 上传文件 | 校验唯一性和必填字段 |
| 预览结果 | 成功、失败、重复、缺字段 |
| 确认导入 | 生成设备记录和入库流水 |
| 导入日志 | 文件名、操作人、结果摘要 |

### 4.3 入库规则

1. 设备码、SN/IMEI/VIN 按类目校验唯一性。
2. 如后续短租设备接入监管锁，锁控接口或初始化状态异常时只能入库为“待完善”，不能进入可租。
3. 入库后默认状态为 `在库可租` 或 `待完善`。
4. 入库照片、采购凭证、授权文件等进入附件中心。

## 5. 调拨与仓库

| 字段 | 类型 | 说明 |
|---|---|---|
| 调出仓库 | 只读/选择 | 当前仓库 |
| 调入仓库 | 选择 | 目标仓库 |
| 调拨原因 | 下拉/文本 | 补货、维修、转门店、盘点调整 |
| 物流/交接方式 | 文本 | 可选 |
| 交接照片 | 图片 | 可选或必填 |
| 接收确认 | 按配置 | 调入方确认后完成 |

规则：

1. 出租中、归还待验设备不允许普通调拨。
2. 调拨中设备不能被短租锁定。
3. 调拨完成后更新当前仓库和设备日志。

## 6. 设备锁定

| 场景 | 说明 |
|---|---|
| 客户扫码短租下单 | 提交订单时锁定可租设备 |
| 门店生成指定设备二维码 | 生成二维码时锁定指定设备 |
| 审核预留 | 审核通过前预留设备 |
| 平台人工锁定 | 高权限用于特殊订单 |

锁定字段：

| 字段 | 说明 |
|---|---|
| 锁定订单 | 订单号 |
| 锁定来源 | 办单助手、客户下单、人工锁定 |
| 锁定人 | 操作人或系统 |
| 锁定截止时间 | 超时释放 |
| 锁定价格方案 | 关联二维码/订单价格快照 |

锁定超时后，如果订单未支付、未审核或已关闭，设备自动释放回 `在库可租`。

## 7. 出库与交付

### 7.1 出库场景

| 场景 | 操作端 |
|---|---|
| 门店当面交付 | 门店手机端 |
| 门店快递发货 | 门店手机端 / 商家 PC |
| 平台仓发货 | 运营端 |
| 资方发货 | 运营端登记或资方端后续开放 |

### 7.2 出库字段

| 字段 | 类型 | 说明 |
|---|---|---|
| 订单号 | 只读 | 关联订单 |
| 设备码 | 选择 | 必填，短租真实设备 |
| 出库仓库 | 只读 | 当前仓库 |
| 发货主体 | 只读/选择 | 门店、商家、平台、资方 |
| 交付方式 | 下拉 | 当面交付、配送、快递 |
| 物流公司/单号 | 文本 | 快递时填写 |
| 设备照片 | 图片 | 必填或按类目 |
| 人机/人车合照 | 图片 | 车辆类建议必填 |
| 配件清单 | 多选 | 电池、充电器、钥匙、包装等 |
| 监管锁接口状态 | 自动/手动 | 如短租设备已接入监管锁 |

短租出库成功后设备状态变为 `出租中`，订单进入待签收或在租，具体按起租规则配置。

## 8. 归还与验收

### 8.1 归还来源

| 来源 | 说明 |
|---|---|
| 客户申请归还 | C 端提交归还 |
| 门店现场归还 | 门店员工扫码处理 |
| 快递归还 | 客户填写物流，平台/门店收货 |
| 租后追回 | 租后或法务找回设备 |

### 8.2 验收字段

| 字段 | 类型 | 说明 |
|---|---|---|
| 设备码 | 选择/扫码 | 必填，短租真实设备 |
| 归还订单 | 关联 | 自动匹配 |
| 外观照片 | 图片 | 必填 |
| 配件照片 | 图片 | 按类目 |
| 监管锁接口状态 | 自动/手动 | 如短租设备已接入监管锁 |
| 验收结果 | 下拉 | 正常、轻微损坏、严重损坏、缺件、设备不一致 |
| 损坏说明 | 文本 | 异常时必填 |
| 赔付金额 | 金额 | 异常时可填 |
| 入库去向 | 下拉 | 可租、维修、争议、下架 |
| 客户确认 | 状态 | 是否需要客户确认验收 |

### 8.3 验收结果

| 结果 | 设备状态 | 后续 |
|---|---|---|
| 正常 | 在库可租 | 订单完成或进入保证金释放 |
| 轻微损坏 | 在库可租/维修中 | 可产生赔付或备注 |
| 严重损坏 | 维修中 | 进入售后/赔付 |
| 缺件 | 争议中/维修中 | 客户补件或赔付 |
| 设备不一致 | 争议中 | 人工复核 |

---

## 9. 维修与下架

| 操作 | 说明 |
|---|---|
| 报修 | 验收异常或日常检查发起 |
| 维修中 | 不可出租 |
| 维修完成 | 可回到在库可租 |
| 报废/下架 | 不再出租 |
| 维修费用 | 可进入财务成本或客户赔付 |

维修记录要关联设备、订单、验收记录和附件。

---

## 10. 盘点

| 功能 | 说明 |
|---|---|
| 发起盘点 | 按仓库、商家、门店、类目 |
| 扫码盘点 | 扫设备码确认在库 |
| 差异列表 | 系统在库但未扫、扫到未登记、状态不一致 |
| 差异处理 | 调整、报损、人工复核 |
| 盘点日志 | 操作人、时间、差异结果 |

盘点不直接删除设备，只能生成差异处理记录。
