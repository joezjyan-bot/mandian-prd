# C 端客户订单详情与账单账单支付

> **Stage 6 术语同步(2026-05-27)**: 本文档已按 Stage 6 统一为商家、联营、平台订单、订单结算款、我的钱包、履约中、逾期费用、留购、保证金等展示术语；数据库字段、API 路径、英文枚举保持不变。

> 页面级 PRD 草案。
> 目标：让客户在小程序/H5/APP 中清楚看到订单进度、待办、合同、账单、账单支付、签收、归还和留购。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 客户订单详情与账单账单支付 |
| 所属端 | C 端小程序/H5/APP |
| 入口路径 | 我的订单 > 订单详情 |
| 使用角色 | C 端客户 |
| 核心目标 | 展示订单状态和下一步待办，并提供支付、授权、合同、公证、绑卡、签收、账单支付、归还、留购入口 |

---

## 2. 页面结构

```
订单详情
├─ 顶部状态与下一步待办
├─ 商品和租赁方案
├─ 费用与账单
├─ 待办任务
├─ 合同/授权/公证
├─ 发货/签收
├─ 账单支付记录
└─ 归还/留购
```

---

## 3. 顶部状态

| 字段 | 说明 |
|---|---|
| 当前状态 | 待付款、待审核、待签约、待发货、在租、逾期、已完成、已关闭 |
| 下一步待办 | 待付款、待授权、待签合同、待绑卡、待确认收货、待支付等 |
| 状态说明 | 用客户能理解的话解释当前卡在哪里 |
| 客服入口 | 联系客服或联系门店 |

客户侧不能展示复杂内部状态，例如后台内部额度、审核人员、平台抽佣细节。

---

## 4. 商品和方案

| 字段 | 说明 |
|---|---|
| 商品 | 商品图、名称、规格、成色、数量 |
| 租赁模式 | 当前展示长租；短租后续需求包接入 |
| 计费单位 | 当前按期/月展示 |
| 租期 | 起租时间、到期时间、期数 |
| 留购 | 是否可留购、留购价格 |
| 增值服务 | 服务名称和金额 |

---

## 5. 费用与账单

| 区域 | 字段 |
|---|---|
| 费用概览 | 首期实付、保证金、总租金、已付金额、剩余应还 |
| 账单列表 | 期数、应还日期、应还金额、已还金额、状态 |
| 逾期信息 | 逾期天数、逾期金额、处理提示 |
| 支付入口 | 立即支付、部分支付、复制支付链接、等待代扣 |
| 支付记录 | 支付时间、金额、方式、状态 |

部分支付后，当期未结清仍展示剩余应还和逾期状态。

| 待授权 | 去授权 |
| 待人脸识别 | 去认证 |

| 待签合同 | 去签署 |
| 待办理公证 | 去办理 |
| 待绑卡 | 去绑卡 |

待办按钮由后台 `next_action` 和链路配置决定。

---

## 7. 合同、授权与公证

| 模块 | 展示 |
|---|---|
| 信用评估/征信授权 | 发起时间、签署状态、再次签署入口 |
| 人脸识别 | 认证状态、失败原因 |
| 主合同 | 合同名称、签署状态、查看/下载 |
| 补充合同 | 补充原因、签署状态 |
| 公证 | 办理状态、费用、办理入口 |
| 代扣绑卡 | 银行卡脱敏、签约状态、解绑规则 |

---

## 8. 发货、签收与归还

| 阶段 | 展示 |
|---|---|
| 待发货 | 发货主体、预计发货说明 |
| 已发货 | 物流公司、单号、配送状态 |
| 门店交付 | 门店信息、交付完成状态 |
| 签收 | 确认收货按钮、电子签收单状态、AI 问答签收视频状态 |
| 归还 | 归还方式、归还地址、物流或到店说明 |
| 验收 | 验收结果、是否有损坏或扣费 |

确认收货方式由后台配置：

1. 普通确认：客户点击确认收货。
2. 电子签收单：客户签署确认签收单合同。
3. AI 问答签收视频：客户按 AI 提问确认本人办理、设备、期数、每期应付金额等，异常回答进入后台人工复核。

---

## 9. 账单支付

| 方式 | 规则 |
|---|---|
| 主动支付 | 客户点击账单支付 |
| 自动代扣 | 已绑卡后按账单日扣款 |
| 部分支付 | 后台生成收款码或客户按规则支付部分金额 |
| 线下支付 | 客户上传或客服登记，财务确认后生效 |

账单支付成功后，客户侧账单、订单状态、支付记录同步更新。

---

## 10. 归还与留购

| 操作 | 规则 |
|---|---|
| 申请归还 | 按订单配置选择到店、快递、上门 |
| 上传凭证 | 可上传快递单或归还照片 |
| 查看验收 | 展示验收结论和争议入口 |
| 申请留购 | 展示留购价和支付入口 |
| 留购完成 | 订单进入已购买或已完成 |

---

## 11. 异常状态

| 异常 | 客户侧提示 |
|---|---|
| 支付处理中 | 请稍后刷新，避免重复支付 |
| 合同失败 | 可重新签署或联系客服 |
| 授权失败 | 可重试或联系客服 |
| 退款中 | 展示退款进度 |
| 订单关闭 | 展示关闭原因摘要 |
| 逾期 | 展示应还金额和账单支付入口 |
