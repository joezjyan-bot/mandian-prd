## 二、关键口径

1. 普通渠道 C 端不展示“留购/买断”作为客户按钮或购买引导,统一展示“申请购买/购买申请”;资方渠道按业务模板保留“提前结清”。
2. 设备价值确认使用订单设备价格作为来源,客户独立勾选确认,并进入订单快照、合同附件和法诉证据包。
3. 到期选择必须三选一同等展示,不得默认购买,不得出现租期满自动转移所有权或自动购买。
4. 提前归还不得默认要求结清全部未到期费用,最终结算以已使用时间、已发生费用、检测验收和实际损失为准。
5. 违约赔偿、设备价值损失、留购/申请购买法律定性需法务书面确认后上线。
