# 资方专属合同模板

> **来源**: P1-4

## 1. 页面定位

运营端 · 资方管理 · 资方详情 · 合同模板页签。配置该资方的专属合同模板,订单分配到该资方时自动切换。

## 2. 业务背景

部分资方(特别是正规金融机构)对合同条款有特殊要求(销售合同/特定条款等)。系统支持每个资方配置专属合同模板,订单分配到该资方时自动切换。

## 3. 主体仍是三方制

无论用默认还是专属模板,合同主体始终是:
- 甲方:门店
- 乙方:客户
- 丙方:平台

资方在合同中**不出现**(避免融资租赁定性)。

仅合同**内容/条款**因资方而不同。

## 4. 配置入口

资方详情 · 合同模板 Tab

## 5. 模板配置项

### 5.1 该资方使用模板的选择
- 选项 1:使用平台默认合同模板(默认)
- 选项 2:使用专属合同模板

### 5.2 专属模板上传
- 上传合同模板文件(.docx 或富文本编辑)
- 设置可变字段占位符(客户姓名、订单金额、设备型号等)
- 预览模板渲染效果
- 测试渲染(用测试数据)
- 保存生效

### 5.3 适用品类(可选)
某些资方专属模板可能针对特定品类:
- 不限品类(所有该资方的订单都用)
- 仅手机品类
- 仅电动车品类
- 多个品类(JSON 数组)

## 6. 切换逻辑(回顾)

```
订单创建后:
- 门店订单 → 默认模板
- 分红/平台订单未分配资方 → 默认模板(尚未到合同环节)
- 分红/平台订单已分配资方 + 资方有专属模板 → 专属模板
- 分红/平台订单已分配资方 + 资方无专属模板 → 默认模板
```

## 7. 版本管理

- 每次专属模板更新生成 v1/v2/v3
- 已签合同的订单按合同签署时的模板版本快照
- 当前版本标记 is_active

## 8. 操作权限

- 运营管理员 + 法务:可上传/修改专属模板
- 普通客服:仅查看
- 资方主体本身:V1 不开放

每次配置/修改:留日志,记操作人、时间、diff。
