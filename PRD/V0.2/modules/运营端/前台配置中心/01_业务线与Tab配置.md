## 4. Tab 字段

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| line_code | 文本(只读) | 是 | 业务线编码,系统固定枚举 |
| display_name | 文本 | 是 | Tab 文案,最长 4 字符 |
| tab_icon | 图标选择器 | 是 | Tabler Outline 图标库;**禁止品类绑定图标**(摩托/单车/手机) |
| tab_icon_active | 图标选择器 | 否 | 选中态图标(默认同 tab_icon) |
| is_enabled | 开关 | 是 | 是否在 App 显示该 Tab |
| tab_order | 数字 | 是 | Tab 顺序(1-4),通过拖拽设置 |
| default_tab | 单选 | 否 | 全局只能有 1 个默认 Tab |
| tab_badge_rule | 下拉 | 否 | 角标规则:无 / 订单待办数 / 客服未读数 / 自定义计算 |

---

## 5. 启停规则

| 场景 | 处理 |
|---|---|
| 关闭"体验租"Tab | C 端立即不显示该 Tab;已存在的体验租订单仍可在"我的-订单中心"查看 |
| 关闭"客服"Tab | 全局客服入口转移到"我的"页和悬浮按钮 |
| 关闭"我的"Tab | **不允许**(系统强制保留) |
| 同时关闭安心用 + 体验租 | **不允许**(必须保留至少 1 个业务 Tab) |

---

## 6. 多租户与城市覆盖

继承 00 总览 §3 三层覆盖架构:

| 层级 | 可配置项 |
|---|---|
| 平台默认 | 全部字段 |
| 租户级覆盖 | 可改:文案、图标、顺序、启停、默认 Tab |
| 城市级覆盖 | 可改:启停(例如某城市暂未开通体验租);其他字段继承租户级 |

**示例**:
- 平台默认:4 Tab 全开
- 鼎租租户:全开
- 鼎租北京:全开
- 鼎租上海:暂时关闭体验租(还没开通门店)

## 7. 操作按钮

| 按钮 | 权限 | 二次确认 | 说明 |
|---|---|---|---|
| 新建覆盖 | 平台管理员 | - | 为指定租户/城市新建覆盖配置 |
| 拖拽排序 | 内容运营 | - | 调整 Tab 顺序 |
| 切换启停 | 内容运营 | ✅ | 影响范围预览 |
| 改图标 | 内容运营 | - | 从 Tabler 图标库选择,禁用品类图标 |
| 设默认 Tab | 内容运营 | - | 全局只能 1 个 |
| 保存草稿 | 内容运营 | - | 不影响线上 |
| 提交审核 | 内容运营 | - | 进入审核流程,见 `08_装修版本与灰度发布.md` |

---

## 8. 失败态与异常

| 异常 | 处理 |
|---|---|
| 提交了不在白名单的图标 | 阻止保存,提示"图标必须来自 Tabler Outline 库,且不绑定品类" |
| 城市级覆盖关闭了租户级的默认 Tab | 阻止保存,提示"该城市无可用默认 Tab,请先设新默认" |
| 多个 Tab 同时设默认 | 阻止保存 |
| 删除业务线编码 | 不允许(只能停用) |
