## V0.1 (2026-05-21) - 初版

### 新增（Added）

PRD 总体大纲首版，覆盖以下内容：

**核心业务模型**
- 三类订单驱动模型（门店订单 / 分红订单 / 平台订单），按出资比例驱动业务逻辑
- 完整订单状态机
- 资金流转模型 + 软钱包账户体系
- 计费规则（搬运惠讯租办单助手）
- 退款 / 续租 / 留购 / 账单日修改流程

**6 端 PRD**
- C 端（小程序 / App）
- 门店端（H5）
- 商家端（PC）
- 运营端（PC）
- 资方端（H5 只读）
- 渠道端（H5 报表）

**后台基础设施**
- IM 客服模块（自研）
- 第三方中控平台
- 配置中心（多层级覆盖）
- 合同模板引擎（出租方可切换）
- 多客户租户与部署架构
- 设备管理与安全锁集成
- 风控引擎

**全局规则与数据模型**
- 全局状态机汇总
- 权限模型（RBAC）
- 全局计费 / 支付 / 推送规则
- 核心实体 ER 图
- 关键表字段定义

**短租预留**
- 长短租差异点分析
- V0.1 架构层面预留扩展点
- V0.2 短租问卷待补清单

**重构注意事项落地**
- 12 条原则的逐条落地方案

**待澄清问题**
- 22 个待澄清问题清单

**实施建议**
- 阶段划分
- 团队配置
- 风险点
