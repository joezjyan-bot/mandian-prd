# 关键表字段索引草案

> 所属：开发设计
> 目标：把数据库表结构草案继续细化到关键字段、唯一约束和高频索引，优先覆盖订单、财务、商品、设备、回调这些核心链路。

---

## 1. 字段约定

所有业务表默认包含：

| 字段 | 说明 |
|---|---|
| `id` | 内部主键 |
| `created_at` | 创建时间 |
| `updated_at` | 更新时间 |
| `deleted_at` | 软删除时间，可空 |
| `created_by` | 创建人，可空 |
| `updated_by` | 更新人，可空 |
| `tenant_id` | 租户或平台客户编号，后续给外部搭建系统使用 |

金额字段统一：

- 金额用分存储。
- 展示时前端格式化为元。
- 不用浮点数保存金额。
