| 项 | 内容 |

### 4.3 工单类型

详见短租 08 §6.1。本页支持的工单类型:

| 类型 | 说明 | 优先级默认 |
|---|---|---|
| `control_api_repeated_failure` | 控车接口连续失败 | urgent |
| `vehicle_offline_long` | 车辆离线 > 1 小时 | high |
| `gps_abnormal` | GPS 异常(漂移/无信号)| normal |
| `unauthorized_zone` | 进入禁行区域(V2)| high |
| `status_inconsistency` | 控车日志与订单状态不一致 | high |
| `customer_complaint` | 客户控车体验投诉 | normal |
| `manual_lockdown_request` | 人工请求锁车 | high |
| `manual_unlock_request` | 人工请求解锁 | high |

### 4.5 工单状态机

| 状态 | 流转 |
|---|---|
| `pending` 待分配 | 系统创建,等待人工认领 |
| `processing` 处理中 | 已被认领,正在处理 |
| `escalated` 已升级 | 普通运营升级到主管 |
| `resolved` 已解决 | 处理完成,待关闭 |
| `closed` 已关闭 | 关闭,归档 |
| `reopened` 重新打开 | 客户反馈未解决,重新进入处理中 |

### 4.6 工单操作

| 操作 | 权限 |
|---|---|
| 认领工单 | 体验租运营 |
| 转派给同事 | 体验租运营 |
| 升级到主管 | 体验租运营 |
| 追加处理记录 | 当前指派人 |
| 标记已解决 | 当前指派人 |
| 关闭工单 | 当前指派人 / 主管 |
| 重新打开 | 主管 |
| 关联其他工单(合并/链接) | 主管 |
| 删除工单 | ❌ 不允许(只能关闭) |

## 6. 性能要求

| 接口 | SLA |
|---|---|
| 控车日志列表 | < 2 秒(默认 100 条/页) |
| 控车日志详情 | < 500ms |
| 工单列表 | < 1.5 秒 |
| 工单详情 | < 1 秒 |
| 中控台状态看板 | < 1 秒(5 分钟缓存) |

控车日志数据量大(预计每日百万级),需做好分库分表和冷热数据分离。

## 7. 数据接口

| 接口 | 用途 |
|---|---|
| `GET /api/v1/admin/exp-rent/control-logs` | 日志列表 |
| `GET /api/v1/admin/exp-rent/control-logs/:id` | 日志详情 |
| `POST /api/v1/admin/exp-rent/control-logs/export` | 导出 |
| `GET /api/v1/admin/exp-rent/exception-tickets` | 工单列表 |
| `GET /api/v1/admin/exp-rent/exception-tickets/:id` | 工单详情 |
| `POST /api/v1/admin/exp-rent/exception-tickets` | 新建工单(手动)|
| `PUT /api/v1/admin/exp-rent/exception-tickets/:id/claim` | 认领 |
| `PUT /api/v1/admin/exp-rent/exception-tickets/:id/transfer` | 转派 |
| `PUT /api/v1/admin/exp-rent/exception-tickets/:id/escalate` | 升级 |
| `POST /api/v1/admin/exp-rent/exception-tickets/:id/comment` | 追加记录 |
| `PUT /api/v1/admin/exp-rent/exception-tickets/:id/resolve` | 标记已解决 |
| `PUT /api/v1/admin/exp-rent/exception-tickets/:id/close` | 关闭 |
| `GET /api/v1/admin/exp-rent/control-channel/health` | 中控台健康 |
