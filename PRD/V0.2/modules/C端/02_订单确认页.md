# 订单确认页

> **Stage 6 术语同步(2026-05-27)**: 本文档已按 Stage 6 统一为商家、联营、平台订单、订单结算款、我的钱包、履约中、逾期费用、留购、保证金等展示术语；数据库字段、API 路径、英文枚举保持不变。

> **来源**: P1-5
> **核心**: 办单二维码扫码后的入口

## 1. 页面定位

C 端 · 客户扫办单二维码后进入此页。锁定门店给定的方案,客户确认后下单。

## 2. 页面布局

### 2.1 顶部
- 门店名称 + 店内地址
- 门店 Logo(如配)
- 客服电话(联系门店按钮)
- 官方反诈提醒入口

### 2.2 商品信息区
- 商品图(主图 + 多图轮播)
- 商品名称
- 商品规格(颜色、容量等)
- 设备型号说明
- 设备款(参考价,非客户实付)
- 服务标签:合同保障、正品保障、客服协助、信用免押(如命中)

### 2.3 规格和套餐区

商品来源分两类：

| 来源 | 客户是否可改 | 说明 |
|---|---|---|
| 办单助手二维码 | 大部分锁定 | 门店已经选择商品、规格、租期和费用，客户只确认 |
| C 端自然流量商品详情 | 按商品配置可选 | 客户可选择规格、长租租期、增值服务 |

可展示字段：

- 规格组：颜色、容量、成色等，来自商品自定义规格。
- 长租租期：按月为主，是否支持天租由配置决定。
- 增值服务：保障服务、配送服务、公证服务等。
- 合同模板：客户侧只展示合同名称和签署主体，不展示内部模板编号。

### 2.4 价格方案区
- 保证金:¥XXX
- 首期金额:¥XXX
- 每期金额:¥XXX × XX 期
- **留购价格(保证金可抵扣)**:¥XXX
- 总金额:¥XXX
- 租期:XX 个月
- 服务费:¥XXX
- 公证费:¥XXX(如有)
- 运费/配送费:¥XXX(如有)

### 2.5 价格双层显示(P1-3)
"实际您要付:¥1000(总金额 ¥1080 - 保证金 ¥80)"
小字辅助说明,避免误解

### 2.6 服务说明
- 服务期内可申请:留购 / 归还 / 续租
- 逾期逾期费用:0.05%/日,最低 10 元
- 客户须知(展开可读)
- 发货和签收方式:物流发货/门店自提/签收合同/签收视频问答
- 反诈提醒:平台不会要求客户向私人账户付款或线下转账

### 2.7 协议勾选

客户提交前必须勾选：

- 租赁服务协议
- 隐私政策
- 个人信息授权书
- 信用评估/信用授权书(如当前链路需要)
- 数字证书使用协议(如当前链路需要)
- 人脸识别服务协议(如当前链路需要)

未勾选时点击主按钮提示：“请先阅读并同意相关协议”。

### 2.8 主 CTA
"立即下单"按钮(大按钮)

### 2.9 二级
- 客服咨询入口
- 商品详情(展开看更多说明)
- 协议查看入口
- 费用明细入口

## 3. 点击"立即下单"

```
1. 检测客户登录状态
   - 未登录 → 拉起支付宝/微信授权登录
   - 已登录 → 继续
2. 检测客户实名状态
   - 已实名 → 跳"资料补充"页(03)
   - 未实名 → 跳"实名认证"页(04)
3. 系统在后台创建订单 draft(状态: pending_info)
   - 关联门店、商品、客户、价格方案、渠道(如有)
   - 固化商品、规格、费用、协议版本、合同模板快照
```

## 4. 异常处理

| 异常 | 提示 |
|---|---|
| 二维码失效 | "此办单链接已失效,请联系门店" + 联系按钮 |
| 商品已下架 | "该商品已下架" |
| 门店暂停 | "门店暂停营业,请联系门店或选其他门店" |
| 客户触发限制办理 | "您当前无法办理此业务,请联系平台客服";不展示具体命中原因 |
| 协议未勾选 | "请先阅读并同意相关协议" |
| 价格快照失效 | "当前价格方案已更新,请重新确认" |
| 合同模板不可用 | "合同模板暂不可用,请联系平台客服" |

## 5. 多端适配

### 5.1 支付宝小程序
- 调用支付宝授权
- 用支付宝芝麻信用做信用评估
- 走直付通做支付/分账(后续)

### 5.2 微信小程序
- 调用微信授权
- 用第三方信用评估
- 走微信支付(V1 不优先)

### 5.3 H5
- 微信内打开:调微信 JSSDK
- 浏览器打开:走通联或直接支付宝/微信 H5 支付

## 6. 客户引导

- 首次进入显示"3 分钟办完租赁,扫脸 → 签合同 → 完成"提示
- 进度条(顶部)显示"1/8 订单确认 → 2/8 实名 → ..."

## 7. 数据模型

订单 draft 创建:
- `order.status = pending_info`
- `order.created_at`
- 关联订单参数(商品、门店、价格方案等)
- `order_snapshot.product`
- `order_snapshot.price_plan`
- `order_snapshot.contract_template_id`
- `agreement_acceptance_records[]`
