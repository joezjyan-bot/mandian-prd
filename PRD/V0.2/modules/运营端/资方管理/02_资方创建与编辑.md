# 资方创建与编辑

> **来源**: P1-2

## 1. 页面定位

运营端 · 资方管理 · 创建资方 / 编辑资方。V1 仅运营线下创建,不开放自助入驻。

## 2. 创建入口

资方列表 · "新增资方"按钮

## 3. 创建表单

### 3.1 基础信息(必填)
- 资方主体名称(公司全称)
- 联系人姓名
- 联系方式(手机号 + 邮箱)
- 合作合同号(平台与资方签订的合作协议号)
- 合作开始日期

### 3.2 偏好配置(必填)
- 可放款品类(多选):手机 / 电动车 / 不限
- 单笔订单设备款上限
- 在租订单累计设备款上限

### 3.3 钱包初始化
- 是否创建钱包:默认是
- 初始充值金额:0

### 3.4 备注(选填)
- 自由文本,承载暂不支持的其他偏好

## 4. 提交逻辑

1. 表单校验
2. 二次确认弹窗
3. 提交后:
   - 创建 funder 记录(status=active)
   - 创建关联钱包
   - 留操作日志
4. 跳转资方详情页

## 5. 编辑

### 5.1 可编辑
- 联系人 / 联系方式 / 合作合同号
- 可放款品类
- 单笔限额 / 总额限额
- 备注

### 5.2 不可编辑
- 资方主体名称(改名走特殊流程)
- 创建人 / 创建时间

### 5.3 限额变更
- 增加:立即生效
- 减少:若当前在租累计已超新限额 → 不允许减
- 留完整变更日志

## 6. 删除 / 停用

- 在租订单不可删除,只可"暂停"或"禁用"
- 暂停/禁用不影响在租订单
- 在租清算后才可"删除"(实际是 status=archived)

## 7. 权限

- 运营管理员 + 财务主管:可创建、编辑、停用
- 普通客服:仅查看

## 8. 操作日志

每次创建/编辑/停用/启用留日志,记操作人、时间、diff。

## 9. 数据模型

见 `00_核心决策表.md` `funder` 表。
