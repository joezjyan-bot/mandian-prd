# 合同公证发起手册模板

> 适用对象：运营客服 / 客户培训 / 商家只读学习

---

## 1. 操作入口

```text
订单详情
  -> 合同 / 公证 / 授权
  -> 发起或重发
```

客户侧：

```text
小程序订单详情
  -> 待办
  -> 合同签署 / 公证办理 / 授权确认
```

---

## 2. 正常流程

```text
运营端发起合同
  -> 客户订单页出现签署待办
  -> 客户完成签署
  -> 第三方回调
  -> 订单详情更新合同状态
  -> 合同附件入附件中心
```

公证流程类似：

```text
运营端发起公证
  -> 客户订单页出现公证待办
  -> 客户完成办理
  -> 回调更新公证状态
  -> 公证材料入附件中心
```

---

## 3. 异常处理

| 异常 | 处理 |
|---|---|
| 合同发起失败 | 检查通道配置，重发 |
| 客户未签署 | 重发待办或联系客服 |
| 回调失败 | 在回调日志重放 |
| 合同过期 | 重新发起 |
| 公证失败 | 查看失败原因，重新发起或人工处理 |

---

## 4. 页面截图占位

1. 订单详情合同状态区。
2. 发起合同按钮和二次确认弹窗。
3. 客户小程序待办页。
4. 合同签署完成页。
5. 回调日志页。
6. 附件中心合同文件页。

---

## 5. 数据联动

- 合同发起生成合同实例。
- 客户待办生成通知。
- 回调进入回调事件表。
- 签署完成更新合同状态和订单状态。
- 合同 PDF 入附件中心。
- 补充合同与改价快照关联。
