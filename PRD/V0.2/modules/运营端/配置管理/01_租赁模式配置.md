# 运营端租赁模式配置

> 页面级 PRD 草案。  
> 目的：统一配置长租、短租、小时租、天租、周租、月租等租期能力，避免在商品或订单里写死。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 租赁模式配置 |
| 所属端 | 运营端，商家 PC 端可按权限引用或维护门店订单配置 |
| 入口路径 | 配置管理 > 租赁模式配置 |
| 使用角色 | 平台管理员、运营配置人员、商家老板 |
| 核心目标 | 配置不同业务场景下可用的租赁模式、计费单位、租期数量、起租和归还规则 |

---

## 2. 核心口径

租期是配置，不是写死字段。

| 层级 | 说明 |
|---|---|
| 租赁模式 | 长租、短租，后续可扩展试租、临租 |
| 计费单位 | 小时、天、周、月 |
| 租期数量 | 例如 2 小时、8 小时、1 天、3 天、1 周、3 月、6 月 |
| 起租规则 | 立即起租、支付后起租、发货后起租、确认收货后起租、门店交付后起租 |
| 归还规则 | 到期归还、续租、留购、买断、超时计费 |

商品和规格只选择适用哪些模式，不直接决定全平台租期规则。

## 4. 配置字段

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 模式名称 | 文本 | 是 | 长租、短租等 |
| 模式编码 | 文本 | 是 | LONG_TERM、SHORT_TERM |
| 是否启用 | 开关 | 是 | 关闭后办单助手不可选 |
| 计费单位 | 多选 | 是 | 小时、天、周、月 |
| 租期数量 | 多选/数字组 | 是 | 每个单位下的可选数量 |
| 起租规则 | 下拉 | 是 | 立即、支付后、发货后、确认收货后、门店交付后 |
| 归还规则 | 多选 | 是 | 到期归还、续租、留购、买断 |
| 超时计费 | 配置 | 短租建议必填 | 超时按小时/天计费 |
| 是否要求绑定设备 | 开关 | 是 | 短租默认是 |
| 适用订单类型 | 多选 | 是 | 门店订单、分红订单、平台订单 |
| 适用类目 | 多选 | 可选 | 手机、电动车、车辆、设备等 |

---

## 5. 默认建议

| 模式 | 计费单位 | 默认租期 | 设备绑定 |
|---|---|---|---|
| 长租 | 月 | 3、6、9、12 | 按类目配置 |
| 短租 | 小时、天、周 | 2小时、8小时、1天、3天、1周 | 必须绑定 |

---

## 6. 与商品配置的关系

商品新增/编辑时读取这里的配置：

1. 选择商品适配长租、短租或两者。
2. 规格可进一步限制适用模式和计费单位。
3. 办单助手只展示商品规格允许且租赁模式配置启用的租期。
4. 短租规格如果没有可用设备库存，不允许生成可下单二维码。

---

## 7. 与订单和账单的关系

订单必须保存配置快照：

| 字段 | 说明 |
|---|---|
| lease_mode | 长租 / 短租 |
| billing_unit | 小时 / 天 / 周 / 月 |
| lease_quantity | 租期数量 |
| lease_start_at | 起租时间 |
| lease_end_at | 到期时间 |
| overtime_rule_snapshot | 超时计费规则快照 |
| return_rule_snapshot | 归还/续租/留购规则快照 |

账单生成必须按计费单位处理，不能默认按月生成。

---

## 8. 操作日志

以下操作必须记录：

- 新增/编辑/停用租赁模式。
- 修改计费单位。
- 修改租期数量。
- 修改起租、归还、超时规则。
- 修改是否要求绑定设备。
- 修改适用订单类型或类目。
