## 工程规范基线

V1 必守的工程规范:

1. 数据库 schema 变更走 migration 框架
2. 代码 Docker 化交付
3. 数据库连接信息走环境变量
4. 任何"租户专属信息"不得硬编码
5. 部署用 CI/CD 流水线 + 灰度发布
6. 全程留操作日志,涉资金动作必须二次确认 + 完整日志
7. 敏感数据默认脱敏
8. 字段命名全部英文,文案走 i18n

详见 `modules/全局/11_工程规范基线.md`
