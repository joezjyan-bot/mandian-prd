# 权限管理

## 模块入口

- 左侧菜单：`权限管理`
- 子菜单：
  - `部门列表`：`/permission/index`
  - `成员管理`：`/permission/member`

## 部门列表

### 入口与路由

- 菜单路径：`权限管理 / 部门列表`
- 路由：`/permission/index`
- 面包屑：`首页 / 权限管理 / 部门列表`

### UI 结构

```text
部门列表
├─ 操作区
│  └─ 添加部门
└─ 表格
   ├─ 部门名称
   ├─ 职能描述
   ├─ 成员数量
   ├─ 添加时间
   └─ 操作
```

### 表格操作

| 点击位置 | 结果 | 风险 |
|---|---|---|
| `添加部门` | 打开添加部门弹窗 | 中 |
| `修改` | 打开修改部门弹窗，带入当前部门名称和职能描述 | 中 |
| `权限设置` | 进入权限设置页 | 高，最终确定会改变权限 |
| `删除` | 未点击 | 高，删除部门前需确认成员归属 |

### 添加/修改部门弹窗

```text
点击 添加部门
└─ 弹窗：添加部门
   ├─ * 部门名称
   ├─ * 职能描述
   ├─ 取消
   └─ 确定

点击 修改
└─ 弹窗：修改部门
   ├─ * 部门名称：带入原值
   ├─ * 职能描述：带入原值
   ├─ 取消
   └─ 确定
```

### 部门权限设置页

```text
点击 权限设置
└─ 页面：权限设置
   ├─ 部门名称
   ├─ 权限设置：全部 / 部分
   ├─ 一级类目
   ├─ 下属二级类目
   ├─ 功能
   ├─ 取消
   └─ 确定
```

#### 权限选择结构

- 左列：一级类目，带复选框和右箭头。
- 中列：选中一级类目后显示二级类目。
- 右列：选中二级类目后显示具体功能权限。
- 点击一级类目右箭头会刷新中列和右列，不会直接保存。
- 点击复选框会改变待保存权限，未点击 `确定` 不应落库。

#### 已见一级类目

`首页`、`商品审核`、`店铺审核`、`订单管理`、`营销管理`、`佣金管理`、`财务管理`、`配置管理`、`权限管理`、`报表查询`、`数据管理`、`服务中心`、`渠道管理`、`信审管理`、`订单列表订单统计`、`租后管理`、`资方管理`、`低费率订单管理`、`分红订单管理`、`采购账户审核`。

#### 示例点击

```text
点击 一级类目：商品审核 右箭头
└─ 中列变为商品审核相关二级类目
   ├─ 商品审核
   ├─ 租赁商品审核
   └─ 库存管理
└─ 右列显示功能
   ├─ 获取商品审核列表
   ├─ 商品审核确认
   ├─ 商品审核详细
   ├─ 查询商品编辑信息
   ├─ 商品信息修改
   └─ 商品审核获取商品类目
```

### 待确认问题

- 权限采用部门权限、成员权限叠加，还是成员权限覆盖部门权限，需要明确。
- `全部` 模式下是否仍允许单独取消某个权限。

## 成员管理

### 入口与路由

- 菜单路径：`权限管理 / 成员管理`
- 路由：`/permission/member`
- 面包屑：`首页 / 权限管理 / 成员管理`

### UI 结构

```text
成员管理
├─ 查询区
│  ├─ 姓名
│  └─ 所属部门
├─ 操作区
│  ├─ 查询
│  ├─ 重置
│  └─ 添加成员
└─ 表格
   ├─ 成员账号
   ├─ 姓名
   ├─ 所属部门
   ├─ 邮箱地址
   ├─ 添加时间
   ├─ 最后登入
   ├─ 是否启用
   └─ 操作
```

### 查询区字段

| 字段 | 类型 | 占位/选项 | 说明 |
|---|---|---|---|
| 姓名 | 输入框 | `请输入姓名` | 按成员姓名查询 |
| 所属部门 | 下拉框 | 已见部门：平台运营部 | 按部门过滤 |

### 按钮与点击反馈

| 点击位置 | 结果 | 风险 |
|---|---|---|
| `查 询` | 刷新表格，成功时 Toast `获取信息成功` | 低 |
| `重 置` | 清空查询条件并刷新 | 低 |
| `添加成员` | 打开添加成员弹窗 | 中，确定会创建账号 |
| `启用/不启用` | 未切换 | 高，影响账号登录 |
| `修改` | 打开修改成员弹窗 | 中 |
| `权限设置` | 进入成员权限设置页 | 高 |
| `删除` | 未点击 | 高 |

### 添加成员弹窗

```text
点击 添加成员
└─ 弹窗：添加成员
   ├─ * 手机号
   ├─ * 成员姓名
   ├─ * 邮箱地址
   ├─ 所属部门：下拉
   ├─ * 登入密码
   ├─ * 确认密码
   ├─ 备注信息
   ├─ 取消
   └─ 确定
```

### 修改成员弹窗

```text
点击 修改
└─ 弹窗：修改成员
   ├─ * 手机号：带入原值
   ├─ * 成员姓名：带入原值
   ├─ * 邮箱地址：带入原值
   ├─ 所属部门：带入原值，可清空/下拉
   ├─ 备注信息
   ├─ 取消
   └─ 确定
```

### 成员权限设置页

```text
点击 权限设置
└─ 页面：权限设置
   ├─ 成员名称
   ├─ 权限设置：全部 / 部分
   ├─ 一级类目
   ├─ 下属二级类目
   ├─ 功能
   ├─ 取消
   └─ 确定
```

- 当前成员权限显示为 `全部`。
- 页面仍展示权限三列结构，未修改任何复选框，点击 `取消` 返回成员列表。

## 权限与安全建议

- 成员账号的手机号、邮箱属于敏感数据，PRD 示例必须脱敏。
- 密码创建、重置、启停账号应记录审计日志。
- 删除成员前需校验是否为最后一个管理员、是否存在未完成审核/财务操作。
- 权限变更建议增加二次确认和变更原因。
