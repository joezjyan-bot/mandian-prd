# 数据看板指标口径

> **Stage 6 术语同步(2026-05-27)**: 本文档已按 Stage 6 统一为商家、联营、平台订单、订单结算款、我的钱包、履约中、逾期费用、留购、保证金等展示术语；数据库字段、API 路径、英文枚举保持不变。

> 页面级 PRD 草案。
> 目标：统一运营、订单、财务、渠道、租后、客诉等指标口径，避免不同页面同一个数字算出来不一致。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 数据看板指标口径 |
| 所属端 | 运营端 |
| 入口路径 | 工作台 > 数据看板 / 数据管理 |
| 使用角色 | 管理员、运营主管、财务、商品运营、渠道运营、租后主管 |
| 核心目标 | 固定指标定义、过滤条件、归属口径和跳转来源，让各模块数据互通可核对 |

数据看板不是先画图表，而是先定指标口径。所有指标都必须能下钻到明细列表。

---

## 2. 核心口径

1. 所有订单指标按 `商家订单 / 联营订单 / 平台订单` 拆分。
2. 金额类指标必须说明按应收、实收、已分账、可提现还是退款后净额统计。
3. 渠道只统计联营订单和平台订单，商家订单默认不进渠道佣金统计。
4. 联营订单按出资比例生成门店份额和资方份额，再分别扣平台抽佣。
5. 订单状态、账单状态、支付状态、合同状态、发货状态不能混用。
6. 看板数字必须和订单列表、财务明细、渠道明细能互相校验。
7. 数据权限要按角色、商家、渠道、资方范围过滤。

---

## 3. 筛选维度

| 维度 | 说明 |
|---|---|
| 时间范围 | 今日、昨日、近 7 天、本月、自定义 |
| 时间类型 | 下单时间、审核时间、支付时间、分账时间、到期时间 |
| 订单类型 | 商家订单、联营订单、平台订单 |
| 商家/门店 | 订单归属 |
| 资方 | 联营订单、平台订单 |
| 渠道来源 | 渠道、推广人、推广码 |
| 商品类目 | 手机、电动车、短租车辆等 |
| 租赁模式 | 长租、短租 |
| 计费单位 | 小时、天、周、月 |
| 下单渠道 | 支付宝、微信、H5、APP、门店扫码 |
| 审核人员 | 审核客服或主管 |
| 租后负责人 | 租后客服 |

---

## 4. 订单指标

| 指标 | 定义 | 下钻 |
|---|---|---|
| 新增订单数 | 时间范围内创建的订单数 | 订单列表 |
| 有效订单数 | 未关闭、未取消的订单数 | 订单列表 |
| 待付款订单数 | 客户未完成首期或前置支付 | 待付款列表 |
| 待审核订单数 | 需要当前端审核的订单 | 待审核队列 |
| 待分配资方数 | 联营/平台订单未绑定资方 | 待分配资方 |
| 待签约数 | 合同/授权/公证未完成 | 待签约列表 |
| 待发货数 | 审核和前置链路完成但未发货 | 待发货列表 |
| 在租订单数 | 已生效且未完成/关闭的订单 | 在租列表 |
| 已完成订单数 | 正常完成、归还、留购完成 | 已完成列表 |
| 已关闭订单数 | 取消、审核拒绝、退款关闭等 | 已关闭列表 |

注意：商家订单的“待审核”指商家自审；联营订单、平台订单的“待审核”指运营端审核。

---

## 5. 金额指标

| 指标 | 定义 |
|---|---|
| 应收租金 | 按订单账单计划生成的租金总额 |
| 实收租金 | 已成功支付并入账的租金 |
| 首期实收 | 首期租金、服务费、公证费等按配置实收 |
| 增值服务实收 | 服务费、设备管理费、公证费等实收 |
| 保证金金额 | 商品保证金、补充保证金、实际冻结金额 |
| 留购金额 | 客户留购/留购应付和实付 |
| 退款金额 | 已退款成功金额 |
| 冲正金额 | 因退款、错账、分账错误产生的冲正金额 |
| 平台抽佣 | 平台按订单规则扣取的佣金 |
| 渠道佣金 | 按渠道规则生成的佣金 |
| 可提现金额 | 满足账期和规则的余额 |

金额指标必须支持查看计算来源：订单、账单期数、支付流水、分账流水、规则版本。

---

## 6. 财务指标

| 指标 | 说明 |
|---|---|
| 待分账金额 | 已收款但未分账 |
| 已分账金额 | 已进入商家/资方/渠道账户 |
| 分账失败金额 | 分账执行失败 |
| 待提现金额 | 各主体已提交提现待审核 |
| 提现成功金额 | 已打款成功 |
| 打款失败金额 | 打款通道失败或账户异常 |
| 对账差异金额 | 系统账和通道账不一致 |
| 冻结金额 | 风控、退款、争议、租后冻结 |

财务指标应和 `钱包、分账、提现与对账` 明细一致。

---

## 7. 商家与商品指标

| 指标 | 说明 |
|---|---|
| 新增商家数 | 提交入驻的商家数量 |
| 审核通过商家数 | 入驻审核通过 |
| 待审核商家数 | 资质、收款账户、授权待审核 |
| 商品总数 | 平台商品 + 商家商品 |
| 待审核商品数 | 商家提交待审核或复审 |
| 已同步办单助手商品数 | 已同步到门店/联营/平台助手 |
| 同步失败商品数 | 商品或规格同步失败 |
| 短租可租设备数 | 设备状态为在库可租 |
| 设备出租率 | 出租中设备 / 可运营设备 |

---

## 8. 渠道指标

| 指标 | 定义 |
|---|---|
| 推广入驻商家数 | 渠道推广码带来的商家入驻数 |
| 入驻通过数 | 渠道推广商家审核通过数 |
| 联营订单数 | 推广商家产生的联营订单 |
| 平台订单数 | 推广商家产生的平台订单 |
| 渠道订单流水 | 联营订单、平台订单实收流水 |
| 待结算佣金 | 已生成但未满足结算条件 |
| 可提现佣金 | 满足结算条件 |
| 已提现佣金 | 提现成功金额 |
| 佣金冲正 | 退款、关闭、逾期冻结导致扣回 |

商家订单默认不计渠道佣金，除非运营端明确配置特殊规则。

---

## 9. 租后指标

| 指标 | 定义 |
|---|---|
| 今日到期 | 今天应还或应归还 |
| 三日到期 | 未来三天应还或应归还 |
| 已逾期订单数 | 至少一笔账单逾期 |
| 逾期金额 | 当前逾期待还金额 |
| 逾期天数分层 | 1-3 天、4-7 天、8-15 天、15 天以上 |
| 催收跟进数 | 已分配并有跟进记录 |
| 回款金额 | 租后期间回收金额 |
| 外部催收同步异常 | 推送或回调失败数量 |

租后指标不能直接用订单到期日代替账单应还日，长租和短租要按不同配置计算。

---

## 10. 客诉指标

| 指标 | 定义 |
|---|---|
| 新投诉数 | 新同步或新创建投诉 |
| 待回复数 | 需要客服回复 |
| 临近超时数 | 距回复截止小于配置阈值 |
| 已超时数 | 超过回复时限 |
| 高风险投诉数 | 涉及扣款、退款、监管、主管关注 |
| 已完结投诉数 | 第三方和本地均完结 |
| 投诉关联退款金额 | 因投诉发起或关联的退款 |

客诉指标应能下钻到支付宝投诉处理列表。

---

## 11. 风控指标

| 指标 | 定义 |
|---|---|
| 风控授权发起数 | 客户风控/征信授权发起次数 |
| 风控报告完成数 | 报告成功生成 |
| 风控异常数 | 授权失败、报告失败、回调失败 |
| 黑名单命中数 | 内部或外部黑名单命中 |
| 审核通过率 | 审核通过订单 / 已审核订单 |
| 审核拒绝率 | 审核拒绝订单 / 已审核订单 |
| 补资料率 | 要求补资料订单 / 待审核订单 |

---

## 12. 数据权限

| 角色 | 数据范围 |
|---|---|
| 平台管理员 | 全平台 |
| 审核客服 | 自己负责或分配范围 |
| 财务 | 财务相关全量，客户敏感信息脱敏 |
| 商家角色 | 仅自己商家数据，运营端看板不开放 |
| 渠道角色 | 仅自己推广商家产生的联营/平台订单 |
| 资方角色 | 后续仅自己资方订单和账单 |
