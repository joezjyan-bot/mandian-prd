# 门店首页与订单中心

> 页面级 PRD 草案。
> 目标：定义门店手机端登录后的首页、办单入口、待办任务、订单中心、员工权限和老板/店员差异。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 门店首页与订单中心 |
| 所属端 | 门店手机端/H5 |
| 入口路径 | 门店登录成功后默认进入 |
| 使用角色 | 商家老板、门店管理员、店长、店员 |
| 核心目标 | 让门店现场人员快速办单、处理待办、查看订单进度，同时限制员工看不到钱包和敏感经营数据 |

---

## 2. 核心口径

1. 门店手机端和商家 PC 端属于同一商家主体，只是使用场景不同。
2. 老板/管理员可看更多经营和钱包入口，店员只看办单助手和待办。
3. 办单助手固定三入口：门店订单、分红订单、平台订单。
4. 门店订单由商家自审；分红订单、平台订单查看平台审核进度。
5. 默认门店发货，所以交付材料、发货、签收、归还验收是首页重要待办。
6. 同一手机登录保持默认 30 天；员工账号建议 7 天，按全局登录策略。

---

## 3. 首页结构

```text
门店首页
├─ 顶部：门店名称、账号角色、消息
├─ 办单助手三入口
│  ├─ 门店订单
│  ├─ 分红订单
│  └─ 平台订单
├─ 待办任务
│  ├─ 待补资料
│  ├─ 待交付材料
│  ├─ 待发货
│  ├─ 待签收
│  └─ 待归还验收
├─ 订单中心
├─ 商品/设备快捷入口
├─ 钱包/收益
└─ 我的
```

员工账号隐藏钱包/收益、商品配置、员工管理等入口。

---

## 4. 顶部信息

| 字段 | 说明 |
|---|---|
| 门店名称 | 当前登录门店 |
| 商家主体 | 所属商家 |
| 当前角色 | 老板、管理员、店长、店员 |
| 消息提醒 | 待办、审核、客诉、发货、归还 |
| 切换门店 | 多门店账号可切换 |

---

## 5. 办单助手入口

| 入口 | 展示条件 | 说明 |
|---|---|---|
| 门店订单 | 商家启用且员工有权限 | 商家自营订单 |
| 分红订单 | 运营端启用且员工有权限 | 门店出设备/部分资金，平台审核 |
| 平台订单 | 运营端启用且员工有权限 | 门店送单给平台，平台审核 |

入口卡片展示：

| 字段 | 说明 |
|---|---|
| 可用商品数 | 当前入口可选商品/规格 |
| 今日出码数 | 当前账号或门店生成二维码数量 |
| 今日下单数 | 客户扫码提交订单数量 |
| 异常提示 | 商品未同步、配置缺失、授权未完成 |

---

## 6. 待办任务

| 待办 | 展示内容 | 操作 |
|---|---|---|
| 待补资料 | 客户或平台要求门店协助补充 | 去处理 |
| 待交付材料 | 设备照、人机/人车合照、配件照 | 上传 |
| 待发货 | 默认门店发货订单 | 发货/交付 |
| 待签收 | 客户未确认收货 | 提醒/查看 |
| 待归还验收 | 客户归还待验 | 验收 |
| 异常 | 材料驳回、设备码异常、监管锁异常 | 查看 |

待办按截止时间和严重程度排序。

---

## 7. 订单中心

### 7.1 分类

| 分类 | 说明 |
|---|---|
| 全部 | 当前权限可见订单 |
| 门店订单 | 商家自营 |
| 分红订单 | 平台审核，门店参与收益 |
| 平台订单 | 平台主控 |
| 待办订单 | 当前需要门店处理 |
| 已逾期 | 当前门店可见逾期 |

### 7.2 筛选

| 字段 | 说明 |
|---|---|
| 订单号 | 精确查询 |
| 客户姓名 | 权限内查询，列表脱敏 |
| 手机号 | 后四位 |
| 商品 | 模糊查询 |
| 办单员工 | 老板/店长可筛选 |
| 状态 | 待审核、待发货、在租、逾期、完成 |
| 时间 | 下单、审核、发货、到期 |

### 7.3 列表字段

| 字段 | 说明 |
|---|---|
| 订单号 | 订单编号 |
| 订单类型 | 门店/分红/平台 |
| 商品 | 商品、规格、租期 |
| 客户 | 脱敏姓名、手机号 |
| 当前状态 | 待审核、待签约、待发货、在租等 |
| 下一步 | 当前门店要做什么 |
| 办单员工 | 实际操作员工 |
| 操作 | 查看、处理待办、联系客服 |

## 8. 订单详情移动端

移动端详情展示重点：

| 区域 | 内容 |
|---|---|
| 状态 | 当前状态和下一步待办 |
| 客户 | 脱敏客户信息、地址摘要 |
| 商品 | 商品、规格、租期、设备码 |
| 费用 | 首期、账单、押金、留购价，员工可按权限隐藏 |
| 审核 | 门店订单可审核，分红/平台只看进度 |
| 发货 | 发货/交付材料 |
| 合同授权 | 客户完成情况 |
| 日志 | 最近操作和备注 |

员工账号不可查看钱包、分账、资方账户、完整敏感资料。

---

## 9. 商品与设备快捷入口

| 入口 | 说明 |
|---|---|
| 可办商品 | 查看当前门店可出码商品 |
| 商品同步异常 | 商品未同步、规格价格缺失 |
| 设备库存 | 短租设备、车辆设备 |
| 扫设备码 | 出库、归还、验收 |

店员默认只可查看可办商品和扫设备码，不可编辑商品配置。

---

## 10. 钱包与收益

仅老板和授权管理员可见。

| 字段 | 说明 |
|---|---|
| 可提现余额 | 商家/门店可提现 |
| 待结算 | 未满足结算条件 |
| 冻结金额 | 争议、退款、租后冻结 |
| 今日入账 | 今日钱包入账 |
| 提现入口 | 跳转提现 |
| 流水明细 | 查看钱包流水 |

店员不展示该模块，接口也必须后端拒绝。

## 11. 我的

| 功能 | 说明 |
|---|---|
| 账号信息 | 当前账号、手机号脱敏、角色 |
| 切换门店 | 多门店账号 |
| 修改密码 | 员工可改自己的密码 |
| 员工管理 | 老板/管理员可见 |
| 登录设备 | 查看当前设备和退出 |
| 协议与帮助 | 服务协议、隐私政策、操作文档 |
| 退出登录 | 清除登录态 |

---

## 12. 通知

| 类型 | 说明 |
|---|---|
| 客户下单 | 办单二维码已提交订单 |
| 审核结果 | 平台审核通过/拒绝/补资料 |
| 待办生成 | 交付、发货、归还 |
| 材料驳回 | 需要重传 |
| 客诉协助 | 需要补充材料 |
| 到期逾期 | 租后提醒 |

## 13. 待确认

2. 门店手机端是否开放提现，还是仅商家 PC 端和老板账号开放。
3. 短租设备扫码是否首页常驻，还是只在待办和设备管理出现。
