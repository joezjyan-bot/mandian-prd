## 仓库定位

本仓库专门存放【满点租赁系统重构】项目的 PRD 文档及评审反馈归档。

- ✅ **源文件存档**（每次修订有 Git 版本历史，可追溯、可回滚）
- ✅ **PRD 单一来源**（飞书是展示层，本仓库是真实来源）
- ✅ **评审反馈归档**（每轮评审的反馈 CSV 归档保留）

代码仓库**另外建立**（mandian-php 或类似命名，重构启动时创建）。

## 协作流程

```
飞书知识库（同事评审 + 评论）
   ↓ 反馈汇总
产品负责人（Hudson）整理反馈
   ↓ 反馈清单
AI 修订（在本 GitHub 仓库修订源文件 + 生成修订对照表 + Changelog）
   ↓ 修订交付物
同步到飞书（替换内容 + 更新 Changelog + 企微通知）
```
