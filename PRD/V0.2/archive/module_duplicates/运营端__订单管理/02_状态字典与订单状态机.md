## 2. 核心三模式命名

| UI 展示名 | `cooperation_mode` | 说明 |
|---|---|---|
| 商家订单 | `self_operate` | 商家自营,商家作为出租方和主要履约方 |
| 联营订单 | `joint_venture` | 平台、商家、资方按规则协作,资方不在 C 端展示 |
| 平台订单 | `receivables_assignment` | 平台主导获客和审核,可分配资方及履约门店 |

**约束**:数据库字段名、API 路径、英文枚举保持不变;页面和文档展示用语统一使用商家订单 / 联营订单 / 平台订单。

## 3. C 端 8 状态

| C 端 Tab | 说明 |
|---|---|
| 全部订单 | 包含所有订单状态 |
| 审核中 | 客户已提交,系统/客服在审核 |
| 待付款 | 客户需支付首期或前置款项 |
| 待签约 | 客户需签署电子合同 / 代扣 |
| 已发货 | 已发出,等待物流送达 |
| 待收货 | 客户已看到物流信息,收到设备后确认 |
| 履约中 | 客户已确认收货,处于订单周期内 |
| 已完成 | 已完成 / 已取消 |

后台状态到 C 端状态的完整映射见 `09_C端订单状态与账单支付.md` v1.2 §1.2。
