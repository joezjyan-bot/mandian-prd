## 2. 核心口径

1. 业务页面只选择“通道能力”，不直接暴露接口密钥、账号、证书等敏感配置。
2. 敏感参数只能放后端安全配置或环境变量，不进入 PRD、前端、普通日志。
3. 通道切换需要按平台默认、订单类型、商家、商品、地区等维度配置。
4. 所有回调必须幂等处理，重复回调只能更新日志，不能重复入账或重复推进状态。
5. 接口异常不直接覆盖人工状态，进入异常队列后由规则或人工处理。

| 短信通道 | 验证码、通知、租后提醒 |
| 监管锁通道 | 锁机、解锁、状态查询 |
| 中控台接口 | 条数购买、报告调用、调用统计 |

## 5. 配置字段

| 字段 | 类型 | 说明 |
|---|---|---|
| 通道编码 | 文本 | 系统唯一 |
| 通道名称 | 文本 | 展示用 |
| 适用范围 | 多选 | 平台默认、订单类型、商家、商品、地区 |
| 支持能力 | 多选 | 支付、退款、签署、查询等 |
| 费用承担方 | 下拉 | 平台、商家、资方、客户 |
| 是否计费 | 开关 | 是否生成接口计费 |
| 超时时间 | 数字 | 接口请求超时 |
| 重试策略 | 配置 | 次数、间隔、退避 |
| 回调地址 | 只读 | 系统生成 |
| 敏感配置引用 | 选择 | 引用后端配置名，不展示值 |

---

## 6. 切换规则

| 场景 | 规则 |
|---|---|
| 平台默认切换 | 影响新订单，历史订单保留通道快照 |
| 指定商家覆盖 | 只影响该商家新订单 |
| 订单异常重试 | 优先使用原通道，切换需人工确认 |
| 通道故障 | 可自动降级到备用通道 |
| 合同已发起 | 不允许随意切换通道，需重发任务 |
| 支付已成功 | 不允许切换原支付通道，只能退款/冲正 |

---

## 7. 回调日志

| 字段 | 说明 |
|---|---|
| 回调编号 | 系统生成 |
| 通道类型 | 支付、合同、公证、物流等 |
| 通道名称 | 来源通道 |
| 业务对象 | 订单、账单、合同、提现、设备 |
| 幂等号 | 通道流水或业务唯一号 |
| 回调状态 | 成功、失败、重复、冲突 |
| 处理结果 | 已推进、已忽略、进入异常 |
| 请求摘要 | 脱敏后的摘要 |
| 响应摘要 | 系统处理摘要 |
| 发生时间 | 回调时间 |
| 操作 | 查看、重放、标记、转异常 |

回调日志不能展示敏感原文，必要时只有技术高权限可查看脱敏前安全日志。

---

## 8. 异常队列

| 异常 | 处理 |
|---|---|
| 回调验签失败 | 拒绝处理，记录安全日志 |
| 重复回调 | 幂等忽略 |
| 金额不一致 | 进入财务异常 |
| 状态冲突 | 进入人工复核 |
| 业务单不存在 | 挂起，支持人工认领 |
| 通道超时 | 按策略重试 |
| 回调缺字段 | 进入技术异常 |

---

## 9. 权限与日志

| 操作 | 权限 |
|---|---|
| 新增通道 | 管理员/技术支持 |
| 启停通道 | 管理员 |
| 修改适用范围 | 管理员/运营配置 |
| 查看回调 | 技术支持、财务按类型 |
| 重放回调 | 技术高权限 |
| 切换支付/打款通道 | 财务主管 + 管理员 |

通道配置变更必须有版本、变更原因、影响范围和回滚入口。
