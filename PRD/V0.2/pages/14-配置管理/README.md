# 配置管理

## 模块入口

- 左侧菜单：`配置管理`
- 子菜单：
  - `分类配置`：`/configure/category/index`
  - `模块扫码配置`：`/configure/ModuleScanConfig`
  - `低费率分佣配置`：`/configure/LowRateRebateConfig`
  - `素材中心`：`/configure/MaterialCenter/index`
  - `公告配置`：`/configure/Notice/index`
  - `常见问题配置`：`/configure/quesConfig`
  - `小程序配置`：`/configure/ShortVersion/index`

## 分类配置

### UI 结构

```text
分类配置
├─ 一级类目 Tab
├─ 操作区
│  ├─ 添加一级类目
│  ├─ 修改一级类目
│  ├─ 删除一级类目
│  └─ 添加二级类目
└─ 二级类目表格
   ├─ ID
   ├─ 二级类目名称
   ├─ 排序
   ├─ 上线时间
   ├─ 状态
   └─ 操作
```

### 一级类目 Tab

- 已见：`智能手机`、`二手手机`、`电动车`、`电脑平板`、`耳机数码`、`手表饰品`、`游戏设备`、`相机摄像`、`演出必备`、`家居设备`。
- Tab 容器有左右切换箭头，后续必须继续检查是否还有隐藏类目。

### 按钮与点击反馈

| 点击位置 | 结果 | 风险 |
|---|---|---|
| `添加一级类目` | 打开添加一级类目弹窗 | 中，确定会新增类目 |
| `修改一级类目` | 预期打开当前一级类目编辑弹窗 | 中 |
| `删除一级类目` | 未点击 | 高，删除类目需确认影响商品 |
| `添加二级类目` | 打开添加二级类目弹窗 | 中 |
| 表格 `添加三级类目` | 未最终保存，仅记录入口 | 中 |
| 表格 `修改` | 预期打开二级类目编辑弹窗 | 中 |
| 表格 `查看三级类目` | 进入或展开三级类目列表 | 低 |
| 表格 `删除` | 未点击 | 高 |

### 添加一级类目弹窗

```text
点击 添加一级类目
└─ 弹窗：添加一级类目
   ├─ * 上架渠道：下拉，示例为某渠道
   ├─ * 类目名称
   ├─ * 支付宝类目
   ├─ * 类目排序
   ├─ 跳转链接
   ├─ 取消
   └─ 确定
```

### 添加二级类目弹窗

```text
点击 添加二级类目
└─ 弹窗：添加二级类目
   ├─ * 所属一级类目：下拉
   ├─ * 类目名称
   ├─ * 类目排序
   ├─ * 上架渠道：下拉
   ├─ 取消
   └─ 确定
```

### 待确认问题

- 分类删除是否限制存在商品的类目。
- `支付宝类目` 是否需要做标准类目映射和校验。
- 跳转链接是否允许外链，是否需要白名单。

## 模块扫码配置

### UI 结构

```text
模块扫码配置
├─ Tab：期数配置
├─ Tab：首付比例
└─ Tab：计算器矩阵
```

### 期数配置

| 区域 | 内容 |
|---|---|
| 按钮 | `新增期数` |
| 表格列 | `期数`、`排序`、`状态`、`操作` |
| 状态控件 | `启用/禁用` 开关 |
| 操作 | `编辑`、`删除` |

已见期数：`1期` 到 `12期`。状态开关为配置项，未切换。

### 首付比例

| 区域 | 内容 |
|---|---|
| 按钮 | `新增首付比例` |
| 表格列 | `首付比例`、`排序`、`状态`、`操作` |
| 操作 | `编辑`、`删除` |

已见比例：`20%`、`30%`、`40%`、`50%`、`60%`。

### 计算器矩阵

```text
计算器矩阵
├─ 列：4期、6期、9期、12期
├─ 行：20%、30%、40%、50%、60%
├─ 单元格：数字输入框
└─ 按钮：批量保存
```

- 单元格当前为费率类数值。
- `批量保存` 未点击，属于全局配置写入。

### 待确认问题

- 矩阵数值的业务含义需明确：服务费率、租金系数、利率还是折扣系数。
- 期数/首付比例禁用后，历史订单是否保持原配置不变。

## 低费率分佣配置

### UI 结构

```text
低费率分佣配置
├─ 新增方案
└─ 表格
   ├─ 方案名称
   ├─ 自有分佣比例
   ├─ 一级分佣比例
   ├─ 二级分佣比例
   ├─ 状态
   ├─ 备注

```

### 新增方案弹窗

```text
点击 新增方案
└─ 弹窗：新增低费率分佣方案
   ├─ * 方案名称
   ├─ * 自有分佣比例
```

- 删除方案需检查是否已被订单或渠道引用。

## 公告配置

### UI 结构

```text
公告配置
├─ Tab：商家中心公告
├─ Tab：商家中心通知
└─ Tab：小程序公告
```

### 商家中心公告

- 字段：
  - `* 滚动条内容`：多行文本。
  - `* 弹窗内容`：富文本编辑器。
- 富文本工具栏包含撤销/重做、字号、行高、字间距、颜色、加粗、斜体、下划线、删除线、上下标、清除样式、缩进、对齐、标题、列表、媒体、链接、水平线、全屏。
- 底部按钮：`删 除`、`确 定`。均未点击。

### 商家中心通知

```text
点击 + 新增通知
└─ 弹窗
   ├─ * 素材ID
   ├─ * 问题ID
   ├─ * 排序
   ├─ 取消
   └─ 确定
```

- 列表为空时展示 `暂无数据`。
- 表格列：`素材ID`、`问题ID`、`排序`、`操作`。

### 小程序公告

- 表格列：`渠道`、`公告内容`。
- `公告内容` 为行内输入框。
- 未编辑，避免影响小程序展示。

## 小程序配置

### 入口页

```text
小程序配置
└─ 表格
   ├─ 渠道/位置
   ├─ 首页：进入
   └─ 我的：进入
```

### 首页配置

```text
首页配置
├─ 基本信息
│  ├─ 渠道：某渠道
│  └─ 位置：首页
├─ 热门搜索关键词
├─ 横幅
├─ 金刚区
├─ 专区标题
├─ 专区主图/副图
├─ 标签商品
└─ 二手榜单
```

#### 热门搜索关键词

- 展示 10 个输入框。
- 点击 `修改` 后输入框可编辑，并出现 `取消`、`确定`。
- 本轮点击 `取消` 恢复。

#### 横幅/金刚区/专区图

- `新增横幅` 弹窗字段：`素材ID`、`跳转链接`、`排序`、`背景色`。
- `金刚区` 表格字段：`素材ID`、`跳转链接`、`标题名`、`排序`、`操作`。
- 各专区图表格字段：`素材ID`、`跳转链接`、`排序`、`操作`。
- `删除` 未点击。

#### 标签商品与二手榜单

```text
点击 新增商品
└─ 商品选择弹窗
   ├─ 查询输入框：请输入
   ├─ 查询
   ├─ 重置
   ├─ 新增：未选择商品时禁用
   ├─ 表格：商品编号、商品名称、店铺名称
   └─ 分页
```

- 商品选择弹窗可分页，列表存在大量商品。
- 未选择商品，未点击最终新增。

### 我的配置

```text
我的配置
├─ 基本信息
│  ├─ 渠道：某渠道
│  └─ 位置：我的
├─ 我的订单
└─ 我的服务
```

#### 我的订单

- 表格列：`素材ID`、`名称`、`跳转链接`、`排序`、`操作`。
- 已见名称：`待付款`、`待审核`、`待发货`、`待收货`、`租用中`、`已逾期`。
- `新增订单` 弹窗字段：`素材ID`、`名称`、`跳转链接`、`排序`。

#### 我的服务

- 表格列：`素材ID`、`名称`、`跳转链接`、`排序`、`操作`。
- 已见名称：`押金`、`收藏夹`、`收货地址`、`优惠券`、`认证管理`、`银行卡`。
- `新增服务` 入口存在，预期与 `新增订单` 模式一致，需二次点击确认字段。

## 配置管理通用风险

- 所有 `确定`、`删除`、`批量保存`、状态开关均会影响线上页面或资金规则，本轮只记录入口，不做最终提交。
- 小程序配置里的素材、跳转链接、公告、FAQ 都应有预览、发布、回滚机制。
