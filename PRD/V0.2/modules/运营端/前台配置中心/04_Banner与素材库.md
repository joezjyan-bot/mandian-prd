## 3. 素材库

### 3.1 图片素材

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| asset_id | 文本(系统) | 是 | 系统生成 |
| asset_name | 文本 | 是 | 素材名称(后台搜索用) |
| file_url | 图片上传 | 是 | 阿里云 OSS / 腾讯云 COS |
| thumbnail_url | 文本(系统) | 是 | 系统自动生成缩略图 |
| dimension | 文本(系统) | 是 | 宽 × 高,自动识别 |
| file_size | 数字(系统) | 是 | KB |
| format | 下拉 | 是 | JPG / PNG / WebP / GIF |
| aspect_ratio | 下拉 | 是 | 16:9 / 4:3 / 1:1 / 3:1(横幅) / 9:16(竖) |
| tags | 多选 | 否 | 标签(活动 / 节日 / 品类:电动车 / 业务线:体验租) |
| copyright_status | 下拉 | 是 | 自有版权 / 已授权 / 待审核 |
| created_by | 关联 | 是 | 上传人 |

**校验规则**:
- 文件大小限制:单张 ≤ 2MB
- 推荐尺寸:横幅 1080×360,主图 1080×608,品类图标 240×240
- 上传时自动压缩到推荐尺寸,保留原图

### 3.2 视频素材

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| asset_id | 文本(系统) | 是 | - |
| asset_name | 文本 | 是 | - |
| file_url | 视频上传 | 是 | 支持 MP4 |
| cover_url | 图片上传 | 是 | 视频封面 |
| duration | 数字(系统) | 是 | 秒 |
| dimension | 文本(系统) | 是 | - |
| file_size | 数字(系统) | 是 | MB(限制 ≤ 20MB) |
| copyright_status | 下拉 | 是 | - |

### 3.3 文案素材

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| copy_id | 文本(系统) | 是 | - |
| copy_name | 文本 | 是 | 文案名称 |
| content | 富文本 | 是 | 文案内容(最长 200 字) |
| copy_type | 下拉 | 是 | Banner 主标题 / 副标题 / 按钮文字 / 反诈话术 / 弹窗文案 |
| tags | 多选 | 否 | - |

**合规校验**:
- 自动检测禁词("违约金/罚息/滞纳金" → "逾期费用"、"买断" → "留购"、"联营/资方/分账"全禁)
- 命中禁词阻止保存,提示具体词

### 3.4 跳转链接素材

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| link_id | 文本(系统) | 是 | - |
| link_type | 下拉 | 是 | 商品详情 / 商品列表 / 体验租门店 / 体验租首页 / 活动页 / 外部网页 / 客服 / 不跳转 |
| target_id | 关联/文本 | 视类型 | 商品 ID / 门店 ID / 活动页 URL / 外链 URL |
| business_line | 下拉 | 是 | 跳转所属业务线(用于跨业务线提示) |
| external_url_whitelist | 系统校验 | 是 | 外链必须在白名单域名内 |

**外部链接白名单(初版)**:
- 满点自有域名(`mandian.com` / `dingzu.com`)
- 微信/支付宝小程序内嵌活动页
- 国家认证机构(司法部 / 公证 / 银保监)

### 4.1 Banner 位置位定义

C 端共有以下 Banner 位:

| slot_key | 位置 | 业务线 | 推荐尺寸 |
|---|---|---|---|
| `home_top` | 通用首页顶部(预留,V1 未启用) | 通用 | 1080×360 |

| `assurance_floor_ad` | 安心用楼层广告位 | 安心用 | 1080×608 |
| `experience_home_top` | 体验租首屏顶部 | 体验租 | 1080×360 |
| `experience_store_top` | 体验租门店详情顶部 | 体验租 | 1080×608 |
| `order_detail_bottom` | 订单详情底部 | 通用 | 1080×360 |

### 4.2 Banner 字段

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| banner_id | 文本(系统) | 是 | - |
| banner_name | 文本 | 是 | Banner 名称(后台展示) |
| slot_key | 下拉 | 是 | 位置位 |
| asset_id | 关联 | 是 | 图片素材 |
| title_copy_id | 关联 | 否 | 标题文案(可叠加在图片上) |
| link_id | 关联 | 是 | 跳转目标 |
| priority | 数字 | 是 | 优先级(同位置同时生效时排序) |
| effective_at | 时间 | 是 | 生效时间 |
| expired_at | 时间 | 是 | 失效时间 |
| target_audience_id | 关联 | 否 | 定向客群,空 = 全部用户(详见 `07_客群分组与精准触达.md`) |

| max_impressions_total | 数字 | 否 | 全站最大曝光次数 |

## 5. Banner 投放规则

### 5.1 优先级与同位置展示

同一 slot_key 同时有多个生效 Banner 时:

1. **客群定向优先**:命中精准客群的 Banner 优先于全量 Banner
2. **优先级数值**:`priority` 数字越大越优先
3. **轮播**:满足条件的 Banner 全部进入轮播(最多 5 张)
4. **超过 5 张时**:按优先级取前 5,其余不展示

### 5.2 时间生效

- `effective_at` < 当前时间 < `expired_at` → 生效
- 生效但已达 `max_impressions_total` → 不再展示
- 已达单用户 `max_impressions_per_user` → 该用户不再看到该 Banner

### 5.3 城市匹配

- `target_cities` 为空 → 全部城市
- 用户所在城市在 `target_cities` 内 → 展示
- 用户未授权定位 → 按平台默认城市判断

## 6. 埋点

| 埋点事件 | 触发时机 | 字段 |
|---|---|---|
| banner_impression | Banner 曝光(进入视口) | banner_id, slot_key, user_id, city, audience_segment |
| banner_click | Banner 被点击 | 同上 + link_type, link_target |
| banner_conversion | Banner 点击后下单 | 同上 + order_id, order_amount |

数据进入运营端的 `Banner 数据看板`(在 V2 开发,V1 仅原始数据可导出)。

## 7. 内容审核

| 流程 | 角色 |
|---|---|
| 上传素材 | 内容运营 / 市场运营 |
| 创建 Banner | 内容运营 / 市场运营 |
| 提交审核 | 内容运营 / 市场运营 |
| 内容审核 | 内容审核(查文案合规、图片版权、跳转目标安全) |
| 发布上线 | 内容审核 / 平台主管 |

**审核重点**:
- 文案合规(无禁词)
- 图片版权(版权说明字段填写)
- 跳转目标:不指向"联营/资方/分账"等内部页面;外链在白名单
- 反诈兼容(不与官方反诈话术冲突)

## 8. 权限与日志

| 动作 | 权限 | 二次确认 | 日志 |
|---|---|---|---|
| 上传素材 | 内容/市场运营 | - | 文件名 + 上传人 |
| 删除素材 | 内容运营(限本人创建)/ 管理员 | ✅ | 仅可删除未被引用的素材 |
| 创建 Banner | 内容/市场运营 | - | 完整 Banner 配置快照 |
| 紧急下线 Banner | 平台管理员 | ✅ | 下线原因 |

**素材删除规则**:被任何已发布或待审核的 Banner / 装修楼层引用的素材,**不允许删除**(显示具体引用方);只有完全无引用且非草稿的素材可删除。
