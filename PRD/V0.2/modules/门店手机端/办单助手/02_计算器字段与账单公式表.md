# 计算器字段与账单公式表

> 所属端：门店手机端 / 办单助手
> 目标：把惠讯租 `phone-rent` 计算器逻辑转成新系统可配置字段。具体公式后续以 `phone-rent` 代码为准，本文件先固定字段、快照和数据流。

---

## 1. 核心口径

1. 办单助手 UI 和计算逻辑参考 `joezjyan-bot/calculator` 里的 `phone-rent`。
2. 新系统后台只负责提供商品、规格、租期、增值服务、费率和规则。
3. 门店订单使用商家端配置。
4. 分红订单、平台订单使用运营端配置。
5. 生成二维码时必须保存价格快照，客户扫码后订单沿用该快照。
6. 客服后续仍可改价，但改价必须生成新价格快照和操作日志。

---

## 2. 三入口配置来源

| 入口 | 商品来源 | 费率来源 | 增值服务来源 | 审核主体 |
|---|---|---|---|---|
| 门店订单 | 商家商品池 | 商家端配置 | 商家端配置 | 商家/门店 |
| 分红订单 | 平台/商家同步商品池 | 运营端配置 | 运营端配置 | 运营端 |
| 平台订单 | 平台商品池 | 运营端配置 | 运营端配置 | 运营端 |

---

## 3. 输入字段

### 3.1 商品字段

| 字段 | 说明 |
|---|---|
| 商品名称 | 展示和订单快照 |
| 类目 | 手机、电动车、设备等 |
| 规格 | 全新、二手、容量、颜色、电池等 |
| 设备指导价 | 计算押金、留购、分红出资基础 |
| 是否短租 | 决定是否必须绑定设备码 |
| 是否同步办单助手 | 控制是否展示 |

### 3.2 租赁字段

| 字段 | 说明 |
|---|---|
| 租赁模式 | 长租 / 短租 |
| 计费单位 | 小时 / 天 / 周 / 月 |
| 租期数量 | 例如 2 小时、3 天、6 期 |
| 起租规则 | 支付后、发货后、签收后、交付后 |
| 首期规则 | 首期实付金额、是否包含服务费 |
| 后续账单规则 | 每期租金、期数、账单日 |

### 3.3 金额字段

| 字段 | 说明 |
|---|---|
| 押金 | 固定金额、比例、免押或风控后调整 |
| 首期实付 | 客户扫码下单需要先支付的金额 |
| 后期月付/期付 | 每期账单金额 |
| 设备管理费 | 可按商品或订单类型配置 |
| 公证费 | 可必选或可选 |
| 服务费 | 可计入首期或账单 |
| 运费 | 默认 0，可配置 |
| 留购价 | 到期购买价 |
| 留购总价 | 租金、押金、留购等口径合计 |

### 3.4 分红字段

分红订单增加：

| 字段 | 说明 |
|---|---|
| 设备价 | 门店填写或从商品指导价带入 |
| 需求配资比例 | 下拉选择 20% - 80% |
| 门店出资比例 | `100% - 需求配资比例` |
| 资方出资金额 | 设备价 * 需求配资比例 |
| 门店出资金额 | 设备价 - 资方出资金额 |
| 分账比例 | 按出资比例锁定 |

## 4. 输出字段

办单助手生成报价时必须输出：

- 商品快照
- 规格快照
- 租赁模式
- 计费单位
- 租期数量
- 押金
- 首期实付
- 每期租金
- 总期数
- 服务费
- 公证费
- 设备管理费
- 运费
- 留购价
- 留购总价
- 分红出资比例
- 二维码有效期
- 价格快照编号
- 配置版本

---

## 5. 账单生成规则

```text
办单助手价格快照
  -> 客户扫码确认
  -> 订单创建
  -> 生成首期支付单
  -> 审核通过后生成账单计划
  -> 客户支付或代扣
  -> 账单结清后触发分账
```

要求：

1. 首期实付和后续账单必须分开存。
2. 增值服务要标记计入首期、计入账单或单独收取。
3. 公证费、服务费、设备管理费必须在后台订单详情展示。
4. 客服改价后要保留原价、改后价、改价原因和操作人。
5. 客户已支付后再改价，必须走补差价、退款或补充合同流程。

---

## 6. 二维码和订单快照

二维码内容不直接暴露价格公式，只包含：

- 报价编号
- 订单类型
- 商家 / 门店
- 渠道归属
- 有效期
- 签名

客户扫码后根据报价编号读取服务端价格快照。

快照失效规则：

- 超过有效期。
- 商品下架。
- 规格停用。
- 配置版本被强制禁用。
- 门店账号被停用。

## 7. 与后台联动

| 数据 | 回传位置 |
|---|---|
| 押金 | 订单价格区、财务账单 |
| 首期实付 | 支付单、订单列表 |
| 后续账单 | 账单计划 |
| 服务费/公证费/管理费 | 订单详情、财务流水 |
| 留购价 | 订单详情、客户留购页 |
| 分红比例 | 待审核、资方分配、分账流水 |
| 价格快照 | 交易快照、操作日志 |
