# 合同、公证与授权配置

> 页面级 PRD 草案。
> 目标：把 e 签宝合同、补充合同、公证、风控/征信授权、商家签章授权等能力独立配置，供订单审核、客户待办和商家入驻调用。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 合同、公证与授权配置 |
| 所属端 | 运营端 |
| 入口路径 | 配置管理 > 合同公证与授权 |
| 使用角色 | 平台管理员、法务/合同配置、运营配置、审核客服、技术支持 |
| 核心目标 | 配置三类订单的合同模板、公证规则、授权节点、发起方式、回调和失败处理 |

该页面是链路配置中心的细分页面，专门处理签署、授权、公证，不把合同规则散落在商品或订单备注里。

---

## 2. 核心口径

1. 门店订单、分红订单、平台订单可使用不同合同模板和签署主体。
2. 分红订单、平台订单默认由运营端主控合同和公证链路。
3. 门店订单可由商家选择启用商家模板或平台提供模板。
4. 商家入驻时的 e 签宝授权签章状态要同步到店铺审核和合同发起。
5. 公证、风控授权、征信授权可作为客户待办发起到小程序/H5/APP。
6. 合同、公证、授权状态必须有回调日志，失败进入异常队列。
7. 合同模板、签署主体、费用承担方变更必须保留版本。

## 4. 合同模板

### 4.1 列表字段

| 字段 | 说明 |
|---|---|
| 模板名称 | 合同展示名称 |
| 模板编码 | 系统唯一 |
| 适用订单类型 | 门店订单、分红订单、平台订单 |
| 适用主体 | 平台、商家、资方、客户 |
| 签署服务商 | e 签宝或其他 |
| 版本 | 当前生效版本 |
| 状态 | 草稿、启用、停用 |
| 最近修改 | 修改人和时间 |
| 操作 | 编辑、发布、复制、停用、查看变量、日志 |

### 4.2 模板字段

| 字段 | 类型 | 说明 |
|---|---|---|
| 模板名称 | 文本 | 必填 |
| 模板编码 | 文本 | 必填 |
| 合同类型 | 下拉 | 主合同、补充合同、授权书、入驻合同 |
| 适用订单类型 | 多选 | 三类订单 |
| 适用租赁模式 | 多选 | 长租、短租 |
| 签署主体 | 多选 | 客户、商家、平台、资方 |
| 模板文件 | 文件/模板 ID | 对接 e 签宝模板 |
| 模板变量 | 配置 | 订单、客户、商品、金额、租期、设备码等 |
| 是否启用 | 开关 | 启用后可被规则引用 |

## 5. 订单类型规则

| 字段 | 门店订单 | 分红订单 | 平台订单 |
|---|---|---|---|
| 是否启用主合同 | 商家可配置 | 默认启用 | 默认启用 |
| 合同主控 | 商家/平台配置 | 运营端 | 运营端 |
| 签署主体 | 客户 + 商家/平台 | 客户 + 商家 + 平台/资方 | 客户 + 平台/资方 |
| 是否自动发起 | 可配置 | 审核通过后可自动 | 审核通过后可自动 |
| 是否需要补充合同 | 可配置 | 可配置 | 可配置 |
| 是否启用公证 | 可配置 | 可配置 | 可配置 |
| 客户待办入口 | 是 | 是 | 是 |

规则：

1. 启用合同但未配置模板时，不允许发布规则。
2. 签署主体缺失签章授权时，不允许自动发起，进入异常或待授权。
3. 合同发起时保存模板版本和变量快照。
4. 合同签署完成后回写订单详情和客户订单详情。

## 6. 商家签章授权

商家入驻后需要完成 e 签宝授权签章，平台才能按授权范围发起商家相关合同。

| 字段 | 说明 |
|---|---|
| 商家名称 | 入驻主体 |
| 授权主体 | 企业/个体户/法人 |
| 授权状态 | 未发起、授权中、已完成、失败、过期 |
| 授权用途 | 入驻合同、租赁合同、补充合同 |
| 授权有效期 | 授权起止时间 |
| 授权文件 | e 签宝返回凭证 |
| 最近回调 | 回调时间和结果 |
| 操作 | 发起授权、重发、查看、停用 |

商家签章授权状态要同步到：

1. 运营端店铺审核。
2. 商家工作台授权提醒。
3. 订单合同发起前置校验。

## 7. 公证规则

| 字段 | 类型 | 说明 |
|---|---|---|
| 是否启用公证 | 开关 | 按订单类型、商家、商品配置 |
| 公证服务商 | 下拉 | 当前接入服务 |
| 触发节点 | 下拉 | 审核通过、合同签署后、支付前、人工发起 |
| 是否自动发起 | 开关 | 默认可人工 |
| 客户办理入口 | 开关 | 发起到客户小程序订单页 |
| 公证费用 | 金额/规则 | 可作为增值服务 |
| 费用承担方 | 下拉 | 客户、商家、平台、资方 |
| 失败处理 | 下拉 | 重试、人工处理、跳过、关闭订单 |

公证发起后要在客户订单详情显示待办，客户完成后回写订单详情。

---

## 8. 风控/征信授权

| 字段 | 类型 | 说明 |
|---|---|---|
| 授权类型 | 多选 | 风控授权、征信授权、流水报告授权 |
| 发起节点 | 多选 | 下单后、审核中、人工发起 |
| 是否需要客户签署 | 开关 | 需要客户确认授权 |
| 授权有效期 | 数字 | 过期后需重新发起 |
| 是否消耗中控台条数 | 开关 | 中控台是外部系统 |
| 费用承担方 | 下拉 | 平台、商家、客户、资方 |
| 失败处理 | 下拉 | 重发、补资料、转人工 |

授权完成后，审核客服可查看授权状态和报告状态；敏感报告按权限控制。

---

## 9. 补充合同规则

| 触发场景 | 说明 |
|---|---|
| 客服改价/改套餐 | 金额变化需要客户确认 |
| 资方变更 | 合同未签或已签后变更资方 |
| 续租 | 新账期和金额 |
| 留购 | 买断/留购确认 |
| 售后争议 | 赔付或补充约定 |
| 门店订单商家补充 | 商家自营合同补充 |

补充合同必须关联原订单、原主合同和触发原因。

---

## 10. 回调日志与异常

| 回调类型 | 说明 |
|---|---|
| 合同发起 | 成功/失败、合同编号 |
| 合同签署 | 签署中、已完成、拒签、过期 |
| 公证发起 | 成功/失败、公证编号 |
| 公证完成 | 完成、失败、取消 |
| 授权发起 | 成功/失败 |
| 授权完成 | 已授权、拒绝、过期 |

异常队列：

| 异常 | 处理 |
|---|---|
| 模板变量缺失 | 阻止发起，提示缺失字段 |
| 签章授权缺失 | 进入商家授权待办 |
| 第三方接口失败 | 重试或人工处理 |
| 回调状态冲突 | 进入异常，不覆盖人工结果 |
| 客户待办过期 | 允许重发或关闭 |

## 11. 权限与日志

| 动作 | 权限 | 日志 |
|---|---|---|
| 新增模板 | 合同配置 | 模板基础信息 |
| 发布模板 | 管理员/法务 | 版本和变量 |
| 停用模板 | 管理员 | 影响范围 |
| 发起商家授权 | 店铺审核/管理员 | 授权主体和结果 |
| 发起合同 | 审核客服/系统 | 订单、模板、版本 |
| 发起公证 | 审核客服/系统 | 订单、费用、服务商 |
| 查看合同 | 有权限角色 | 查看日志 |
| 下载合同 | 高权限 | 下载日志 |
| 回调重试 | 技术/管理员 | 重试次数和结果 |
