# 设备列表

> **来源**: P1-1

## 1. 页面定位

运营端 · 库存设备管理 · 设备列表。运营客服查看所有租户的设备(按租户隔离)/ 按门店查看设备。

## 2. 默认筛选器

**已下架设备不显示**(默认排除 status=已下架)

如需查看已下架设备:
- 切换到"已下架设备"视图(单独 Tab)
- 或在筛选器手动加上"已下架"

## 3. 筛选

- 设备 ID / 设备序列号 / IMEI(手机)
- 设备类型 / 型号 / 配置
- 所在门店
- 状态(9 个,可多选)
- 入库时间范围
- 关联订单
- 当前客户(出租中时)

## 4. 列表字段

- 设备 ID
- 设备类型 + 型号 + 配置(如"iPhone 15 / 256GB / 黑")
- 序列号 / IMEI(脱敏部分位)
- 所在门店
- 当前状态(标签 + 颜色)
- 当前关联订单(可跳)
- 当前客户(出租中时,姓名脱敏)
- 入库时间
- 累计出租次数
- 操作:详情 / 状态变更 / 维修记录 / 下架

## 5. 设备详情

进入详情后展示:
- 基础信息(型号 / 配置 / 序列号 / IMEI)
- 入库信息(供货商 / 入库价 / 入库时间)
- 当前状态 + 历史状态(timeline)
- 历史订单列表
- 维修记录
- 监管锁状态(如适用)
- 操作日志

## 6. 状态变更操作

针对单个设备:
- 入库登记(填供货商、入库价、配置)
- 取消保留 / 解锁
- 标记维修(填维修原因)
- 修好后标记可租
- 标记争议
- 下架(必填下架原因 + 处置方式)

各操作二次确认 + 留日志

## 7. 批量操作

支持批量:
- 批量入库登记(扫码/Excel 导入)
- 批量下架(谨慎,需多次确认)
- 批量更新维修状态

## 8. 数据导出

- 支持筛选后导出 Excel
- 字段可选
- 大量数据(>1万)异步导出 + 邮件通知

## 9. 权限

- 运营管理员:全部
- 库存管理客服:全部
- 普通客服:仅查看
- 门店:仅查看本店设备(在商家 PC / 门店手机端)

## 10. 数据模型

见 `00_核心决策表.md` `device` / `device_status_log` 表。
