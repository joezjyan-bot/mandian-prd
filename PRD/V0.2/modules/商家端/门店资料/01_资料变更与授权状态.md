# 资料变更与授权状态

> 页面级 PRD 草案。
> 目标：商家可以在商家端查看入驻资料、授权状态、收款账户和审核进度；需要变更关键资料时提交运营审核，而不是直接覆盖线上资料。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 资料变更与授权状态 |
| 所属端 | 商家 PC 端、门店手机端老板账号 |
| 入口路径 | 商家端 > 店铺资料 / 我的 > 店铺资料 |
| 使用角色 | 商家老板、商家管理员 |
| 核心目标 | 查看资料、提交变更、完成授权、查看审核结果 |

---

## 2. 核心口径

1. 入驻审核通过后的资料形成生效版本。
2. 主体资料、法人资料、收款账户、门店地址、经营类目变更必须提交运营审核。
3. 审核中的变更不影响当前订单履约，除非运营暂停店铺。
4. 收款账户非本人时必须补授权材料。
5. e 签宝签章授权、收款账户授权、接口计费协议状态要对商家可见。

---

## 3. 资料页结构

```text
店铺资料
├─ 基础资料
├─ 营业执照
├─ 法人身份证
├─ 门店地址与照片
├─ 收款账户
├─ 授权状态
├─ 变更记录
└─ 审核进度
```

---

## 4. 可查看资料

| 模块 | 字段 |
|---|---|
| 基础资料 | 店铺名称、联系人、联系电话、经营类目 |
| 营业执照 | 主体名称、信用代码脱敏、法人、注册地址、有效期 |
| 法人身份证 | 姓名、证件号脱敏、有效期 |
| 门店地址 | 文本地址、地图定位、门头照片 |
| 收款账户 | 收款方式、账户名、账号脱敏、实名状态 |

商家端只能查看自身资料，不展示审核内部备注和风控标签明细。

---

## 5. 资料变更申请

| 字段 | 说明 |
|---|---|
| 变更类型 | 基础资料、证照、法人、地址、收款账户、经营类目 |
| 原资料 | 当前生效值 |
| 新资料 | 商家填写 |
| 附件 | 营业执照、身份证、门头照片、授权材料等 |
| 变更原因 | 必填 |
| 联系人 | 默认老板账号 |

提交后：

1. 生成资料变更单。
2. 状态为待审核。
3. 运营端店铺详情收到待办。
4. 审核通过后新版本生效。
5. 审核驳回后保留原因，商家可重新提交。

---

## 6. 授权状态

| 授权项 | 状态 |
|---|---|
| e 签宝签章授权 | 未发起、授权中、已完成、失败、过期 |
| 收款账户授权 | 无需授权、待提交、审核中、已通过、驳回 |
| 接口计费协议 | 未确认、已确认、需重新确认 |
| 合同模板确认 | 未确认、已确认、需更新 |
| 风控能力开通 | 未开通、已开通、停用 |

操作：

| 操作 | 说明 |
|---|---|
| 去授权 | 跳转授权流程 |
| 重新授权 | 失败或过期时使用 |
| 查看凭证 | 查看授权文件摘要 |
| 联系平台 | 授权异常时联系客服 |

---

## 7. 审核进度

| 状态 | 说明 |
|---|---|
| 草稿 | 未提交 |
| 待审核 | 已提交运营 |
| 待补资料 | 运营要求补充 |
| 已通过 | 新资料生效 |
| 已驳回 | 展示驳回原因 |
| 已取消 | 商家主动取消变更 |

每条变更记录展示：提交时间、变更类型、审核状态、审核结果摘要、生效时间。

---

## 8. 权限

| 操作 | 老板 | 商家管理员 | 员工 |
|---|---|---|---|
| 查看店铺资料 | 是 | 可配置 | 否 |
| 提交资料变更 | 是 | 可配置 | 否 |
| 修改收款账户 | 是 | 否/可配置 | 否 |
| 发起授权 | 是 | 可配置 | 否 |
| 查看审核进度 | 是 | 可配置 | 否 |

员工账号不进入资料变更与授权页面。
