## 2. 阅读顺序建议

### 2.1 第一次接触短租业务

按顺序读 00 → 01 → 02 → 05,建立完整业务+数据模型理解。

### 2.2 开发实施

先读 05(数据模型);再按业务领域细读 06-10 各专题:
- 用户 → 06 骑行人
- 资金 → 07 保证金 + 10 逾期费用 + 05 §5 资金事件
- 设备 → 05 §4.3 设备双表 + 08 控车日志
- 订单 → 02 流程 + 05 状态机 + 09 续租链
- SLA → 10 接单 SLA + 03 商家端动作

### 2.3 合规审查

重点看:
- 00 体验租业务边界 — 业务清晰度
- 06 骑行人 — 实名/年龄/驾照/隐私
- 07 保证金 — 扣款合规分级
- 10 逾期 — 逾期费用公式、逾期费用封顶、客户告知
