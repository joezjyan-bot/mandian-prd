## 4. 楼层模块字段

### 4.1 楼层通用字段

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| floor_id | 文本(系统) | 是 | 楼层 ID |
| floor_type | 下拉 | 是 | banner / product_showcase / product_waterfall / image_ad / video / text_link |
| floor_name | 文本 | 是 | 楼层名称(后台展示用) |
| sort_order | 数字 | 是 | 楼层顺序 |
| is_enabled | 开关 | 是 | 是否启用 |
| effective_at / expired_at | 时间 | 否 | 楼层生效/失效时间(用于活动) |
| target_audience | 客群选择器 | 否 | 定向客群,空=全部用户 |

### 4.2 楼层类型详细字段

**banner 轮播**:
- banner_list:Banner 列表(最多 5 张,详见 `04_Banner与素材库.md`)
- auto_play_interval:轮播间隔(秒,默认 4)
- show_indicator:是否显示分页指示器

**product_showcase 商品橱窗(精选)**:
- showcase_title:模块标题(如"新品推荐"/"本周热门")
- showcase_subtitle:副标题(可选)
- product_ids:精选商品 ID 列表(运营手动选,最多 12 个;V2 接算法推荐)
- layout:展示样式(横向滑动 / 2 列 / 3 列)
- show_more_link:是否展示"查看更多"链接

**product_waterfall 商品瀑布流**:
- title:模块标题
- filter_rule:筛选规则(品类/品牌/价格区间/标签)
- sort_rule:排序(默认 / 销量 / 价格 / 最新)
- max_count:最多展示数(默认 20,V1 不做分页)

**image_ad 图片广告位**:
- image_url:图片地址(素材库选)
- aspect_ratio:宽高比(限定 16:9 / 4:3 / 1:1 / 3:1 横幅)
- link:跳转目标(详见 §6 跳转规则)

**video 视频位**:
- video_url:视频地址(素材库选)
- cover_url:封面图
- auto_play:是否自动播放(默认关闭,合规要求)
- mute:是否静音(默认开,合规要求)

**text_link 文字链**:
- text:文字内容(最多 12 字符)
- link:跳转目标

## 5. 品类宫格

| 字段 | 类型 | 说明 |
|---|---|---|
| category_id | 关联 | 引用品类管理的品类(详见 `05_品类导航管理.md`) |
| display_name | 文本 | C 端显示名称(可覆盖品类的标准名,默认继承) |
| icon | 图标上传 | 品类小图标(40x40 推荐) |
| sort_order | 数字 | 顺序 |
| is_enabled | 开关 | 是否在 C 端展示 |
| show_label | 文本(可选) | 角标文字(如"NEW"/"热门") |
| show_label_color | 颜色 | 角标颜色 |

宫格布局:V1 固定 4 列 × N 行;V2 支持 5 列。

## 6. 跳转规则(所有楼层共享)

| 跳转类型 | 必填字段 |
|---|---|
| 商品详情 | product_id |
| 商品列表 | category_id 或 filter_rule |
| 体验租门店详情 | store_id |
| 体验租首页 | - |
| 活动页 | activity_url |
| 外部网页(站外) | external_url(需在白名单内) |
| 客服 | - |
| 不跳转(纯展示) | - |

**注意**:跳转到体验租的链接,**前提是当前用户所在城市已开通体验租**(店铺管理决定);若未开通,跳转时提示"该城市暂未开通体验租"。

## 7. 商品列表装修(简化版,V1)

| 模块 | 配置 |
|---|---|
| 筛选器 | 品类(必选)/ 品牌(可选)/ 价格区间(可选)/ 成色(可选)/ 期数(可选)/ 服务标签(可选) |
| 排序 | 默认 / 销量降序 / 价格升序 / 价格降序 / 最新上架 |
| 列表卡片 | 商品图 / 标题 / 月供起价 / 期数标签 / 服务标签 / 库存状态 |
| 空状态 | 当筛选无结果时的文案与插画 |

## 8. 商品详情装修(模板级)

不针对单个商品,而是定义安心用全部商品**详情页的模板**:

| 模块 | 配置 |
|---|---|
| 价格区分层 | 首期(大字)/ 月供 × 期数(中)/ 留购价(小灰) |
| 服务保障 | 选 4-6 个 ICON + 文字(正品/保修/客服/反诈) |
| 反诈提醒条 | 顶部固定一行(可关闭)/ 文案可改 |
| 租赁须知 | 折叠展开 / 文案运营可改 |
| 底部 CTA | "立即租赁"(主橙)/ "联系客服"(描边) |
