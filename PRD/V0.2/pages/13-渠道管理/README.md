# 渠道管理

## 模块入口

- 左侧菜单：`渠道管理`
- 子菜单：
  - `渠道分佣列表`：`/channels/index`
  - `佣金结算管理`：`/channels/commission`
  - `渠道订单明细`：`/channels/orderNumber`

## 渠道分佣列表

### 入口与路由

- 菜单路径：`渠道管理 / 渠道分佣列表`
- 路由：`/channels/index`
- 面包屑：`首页 / 渠道管理 / 渠道分佣列表`

### UI 结构

```text
渠道分佣列表
├─ 查询区
│  ├─ 渠道名称
│  └─ 分佣状态
├─ 操作区
│  ├─ 查询
│  └─ 新增
└─ 表格区
   ├─ 渠道id
   ├─ 渠道编码
   ├─ 渠道名称
   ├─ 分佣比例
   ├─ 分佣状态
   ├─ 渠道账号
   ├─ 创建时间
   └─ 操作
```

### 查询区字段

| 字段 | 类型 | 占位/选项 | 说明 |
|---|---|---|---|
| 渠道名称 | 输入框 | `请输入` | 按渠道名称模糊查询 |
| 分佣状态 | 下拉框 | `待审核`、`审核通过`、`审核拒绝` | 按渠道审核状态过滤 |

### 按钮与点击反馈

| 点击位置 | 结果 | 风险 |
|---|---|---|
| `查 询` | 刷新表格，成功时顶部 Toast `获取信息成功` | 低 |
| `新 增` | 打开 `新增` 弹窗 | 中，最终 `确定` 会创建渠道 |
| 表格 `查看渠道码` | 打开 `渠道编码` 弹窗，展示二维码，底部 `取 消`、`确 定` | 低 |
| 表格 `修改` | 预期打开渠道编辑弹窗；本轮因操作单元格中 `查看渠道码/修改` 命中区域较近，需二次精确复核 | 中，最终 `确定` 会更新渠道 |
| 表格 `删除` | 未点击 | 高，删除前必须人工确认 |

### 新增弹窗

```text
点击 新增
└─ 弹窗：新增
   ├─ * 渠道名称：请输入渠道名称
   ├─ * 分佣比例：数字步进器，单位 %
   ├─ * 支付宝姓名
   ├─ * 支付宝账号
   ├─ * 渠道账号
   ├─ 初始密码：888888
   ├─ 取消
   └─ 确定
```

### 渠道编码弹窗

```text
点击 查看渠道码
└─ 弹窗：渠道编码
   ├─ 二维码图片
   ├─ 取消
   └─ 确定
```

### 空状态与分页

- 有数据时展示渠道列表。
- 无数据时应展示 `暂无数据`。
- 当前表格有分页组件，需支持总数、上一页、下一页、每页条数。

### 权限与审计

- 新增、修改、删除渠道应写入操作日志。
- 分佣比例变更属于资金规则变更，建议记录变更前后值、操作人、操作时间、原因。

### 待确认问题

- `修改` 链接需在后续复测中精确点击，确认是否与新增弹窗字段完全一致。
- 渠道创建是否需要审核流、是否允许同名渠道、渠道账号是否唯一，需要产品确认。

## 渠道订单明细

### 入口与路由

- 菜单路径：`渠道管理 / 渠道订单明细`
- 路由：`/channels/orderNumber`

### UI 结构

```text
渠道订单明细
├─ 查询区
│  ├─ 渠道名称
│  ├─ 手机号
│  ├─ 订单状态
│  └─ 下单时间
├─ 操作区
│  ├─ 查询
│  └─ 导出
└─ 表格区
   ├─ 店铺名称
   ├─ 订单编号
   ├─ 渠道名称
   ├─ 商品名称
   ├─ 已支付期数/总租期
   ├─ 总租金
   ├─ 已付租金
   ├─ 下单人姓名
   ├─ 下单人手机号
   ├─ 下单时间
   ├─ 订单状态
   └─ 渠道结算金额
```

### 查询区字段

| 字段 | 类型 | 说明 |
|---|---|---|
| 渠道名称 | 输入框 | 按渠道名称筛选 |
| 手机号 | 输入框 | 按下单人手机号筛选 |
| 订单状态 | 下拉框 | `待支付`、`支付中`、`已支付申请关单`、`待审核`、`待终审`、`待发货`、`待确认收货`、`租用中`、`待归还`、`待结算`、`结算待支付`、`订单完成`、`交易关闭` |
| 下单时间 | 日期范围 | 开始时间到结束时间 |

### 按钮与点击反馈

| 点击位置 | 结果 | 风险 |
|---|---|---|
| `查 询` | 刷新表格；无匹配时展示 `暂无数据` | 低 |
| `导 出` | 未点击；应生成导出任务或下载文件 | 高，导出含订单与手机号，必须二次确认 |

### 滚动与隐藏内容检查

- 表格字段较多，应支持横向滚动查看最右侧 `渠道结算金额`。
- 导出结果应进入 `数据管理 / 导出数据下载` 跟踪。

### 权限与审计

- `导 出` 必须独立权限控制，并记录导出条件、操作人、导出时间。
