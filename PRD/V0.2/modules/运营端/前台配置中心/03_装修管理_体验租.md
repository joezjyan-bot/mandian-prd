## 4. 楼层类型(体验租特有)

继承 `02_装修管理_安心用.md` §4 楼层通用字段,以下是体验租特有的楼层类型:

### 4.1 view_mode 视图模式

| 字段 | 类型 | 说明 |
|---|---|---|
| default_view | 下拉 | `list`(列表)/ `map`(地图)。**默认 `list`** |
| show_toggle | 开关 | 是否显示模式切换按钮 |
| toggle_position | 下拉 | 切换按钮位置:顶部右 / 顶部中 / 浮动右下 |

**P0 决策(2026-05-25)**:V1 默认列表模式 + 顶部突出"地图模式"按钮。

### 4.2 store_list 门店列表

| 字段 | 类型 | 说明 |
|---|---|---|
| card_style | 下拉 | `standard`(标准卡)/ `compact`(紧凑卡) |
| sort_rule | 下拉 | `distance`(距离升序,默认)/ `stock`(库存优先)/ `business_status`(营业中优先) |
| max_count | 数字 | 列表上限(默认 50,超过分页) |
| show_stock_summary | 开关 | 是否在门店卡显示库存摘要(可租 X 辆) |
| show_featured_models | 开关 | 是否在门店卡显示主推车型(最多 2 个) |
| show_distance | 开关 | 是否显示距离(必须有定位授权) |
| show_business_hours | 开关 | 是否显示营业时间 |

### 4.3 map_view 地图视图

| 字段 | 类型 | 说明 |
|---|---|---|
| default_zoom | 数字 | 默认缩放级别(11-15,推荐 13) |
| pin_style | 下拉 | Pin 样式(自家品牌色 vs 通用红) |
| cluster_enabled | 开关 | 门店密集时是否聚合(V1 推荐 开) |
| show_my_location | 开关 | 显示用户当前位置(蓝点) |
| info_window_layout | 下拉 | 点击 Pin 后弹出的卡片样式:标准卡 / 紧凑卡 |

### 4.4 category_tabs 品类标签栏

| 字段 | 类型 | 说明 |
|---|---|---|
| layout | 下拉 | `horizontal_scroll`(横向滚动)/ `fixed_grid`(固定网格) |
| categories | 数组 | 引用品类管理(`05_品类导航管理.md` 体验租品类) |
| show_main_tag | 开关 | 是否在主打品类上加 ⭐"主打"角标 |

## 5. 门店详情装修(2.2)— 含长租/短租 Tab ⭐

**P0 决策(2026-05-25)**:进入门店之后,客户可以切"长租 / 短租"两个 Tab 看该门店在两条业务线的内容。

```text
┌─────────────────────────────────────┐
│  ←  选择车辆          ⋯ ⊙           │
├─────────────────────────────────────┤
│  门店头图轮播                        │
│  鼎租新街口店  ⭐4.8                  │
│  📍 朝阳区... | 🕐 营业中            │
│  [联系商家]  [导航到店]               │
├─────────────────────────────────────┤
│  ┌─────────┬─────────┐             │
│  │ 短租 ●  │ 长租 ○  │← Tab          │
│  └─────────┴─────────┘             │
│  Tab 默认值:从体验租进默认短租;       │
│              从安心用进默认长租         │
├─────────────────────────────────────┤
│  [当前 Tab 的车型/商品列表]           │
└─────────────────────────────────────┘
```

| 字段 | 类型 | 说明 |
|---|---|---|
| show_long_short_tab | 开关 | 是否显示长租/短租 Tab |
| default_tab_from_experience | 单选 | 从体验租入口进入时默认 Tab(默认"短租") |
| default_tab_from_assurance | 单选 | 从安心用入口进入时默认 Tab(默认"长租") |
| empty_state_text | 文本 | 当门店未开通对应业务时的提示(如"该门店暂未开通长租业务") |

**门店是否开通对应业务的判定**:
- 体验租 Tab:门店在「店铺管理 > 业务线勾选」中勾了"体验租商家"
- 长租 Tab:门店在「店铺管理 > 业务线勾选」中勾了"安心用商家"
- 都开通 → 显示双 Tab
- 只开通一条 → 隐藏 Tab(直接展示该业务)
- 都未开通 → 不应进入此页(异常情况,降级到城市未开通)

详见 `modules/运营端/店铺管理/0X_商家入驻业务线勾选.md`。

## 7. 订单确认页装修(2.4)— 模板级

不针对单个订单,而是定义体验租所有订单的确认页模板。

| 模块 | 配置 |
|---|---|
| 商品摘要卡 | 车型图 + 标题 + 卖点(1 行) |
| 租车方案区 | 显示长短租 Tab(若车型支持)+ 套餐网格 + 实时合计 |
| 取还信息块 | 取车时间(可选预约)+ 还车时间(系统算)+ 门店信息 |
| 保障包(可选) | 基础版免费(意外医疗/伤残)+ 可加购的升级版 |
| 协议勾选区 | 必勾协议清单(运营可配)+ 隐私 + 反诈 |
| 底部 CTA | "确认订单"(主橙) |

---

## 8. 跨业务线的特殊处理

| 场景 | 处理 |
|---|---|
| 客户从安心用入口进体验租门店 | URL 带 `from=assurance_rent`,门店详情默认进长租 Tab |
| 客户从体验租入口进同一门店 | URL 带 `from=experience_rent`,门店详情默认进短租 Tab |
| 客户切换 Tab 后下单 | 订单 `business_line` 字段以**实际下单时所在的 Tab** 为准 |
| 客户从体验租切到长租 Tab 后下单 | 订单进入安心用长租链路(订单类型 / 状态机 / 履约都按长租) |
