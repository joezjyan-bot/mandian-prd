# 黑名单匹配与审核提示

> 页面级 PRD 草案。
> 目标：建立内部黑名单库和外部黑名单接口预留能力，在客户下单和审核时进行脱敏匹配，命中后给审核人员明确提示。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 黑名单匹配与审核提示 |
| 所属端 | 运营端 |
| 入口路径 | 风控管理/黑名单库 |
| 使用角色 | 风控、审核客服、客服主管、平台管理员 |
| 核心目标 | 管理脱敏黑名单数据，支持下单匹配、审核提示、加黑、解除、日志追溯和外部库对接 |

黑名单库先作为平台风控预警，不直接自动拒单。命中后应在审核时弹出“疑似黑名单用户，请严格审核”，由审核人员结合资料判断。

---

## 2. 核心口径

1. 黑名单存储脱敏姓名、脱敏手机号、脱敏身份证号、命中标识和来源，不在普通列表展示完整敏感信息。
2. 客户下单、资料提交、审核打开订单时都应触发匹配。
3. 命中后进入订单风险提示，不直接替代人工审核。
4. 支持内部黑名单、平台手工加黑、外部黑名单库接口三种来源。
5. 加黑、解除、查看敏感信息都必须记录日志。
6. 黑名单匹配结果要写入订单风控快照，后续黑名单库变化不改变历史审核依据。
7. V1 可先做内部库和预留接口；外部黑名单库后期对接。

| 手机号 | 文本 | 可选，存储脱敏展示和匹配索引 |

## 5. 匹配规则

### 5.1 匹配触发点

| 触发点 | 说明 |
|---|---|
| 客户扫码提交手机号 | 初步手机号匹配 |
| 实名认证完成 | 姓名 + 身份证匹配 |
| 银行卡/代扣授权完成 | 可按实名信息再次匹配 |
| 审核客服打开订单 | 拉取最新内部库和外部库结果 |
| 订单改资料 | 客户资料变化后重新匹配 |
| 批量风控任务 | 对待审核订单批量复查 |

### 5.2 匹配优先级

| 优先级 | 规则 | 结果 |
|---|---|---|
| 高 | 身份证号完全匹配 | 强提示 |
| 高 | 姓名 + 手机号匹配 | 强提示 |
| 中 | 手机号匹配 | 疑似提示 |
| 中 | 姓名 + 地址/设备/历史订单相似 | 疑似提示 |
| 低 | 外部库模糊命中 | 提示复核 |

匹配结果要保存命中规则、命中来源、命中时间和命中证据摘要。

## 6. 审核提示

### 6.1 列表提示

在待审核、订单列表、订单详情中展示风险标签：

| 标签 | 展示规则 |
|---|---|
| 疑似黑名单 | 中低风险命中 |
| 黑名单命中 | 高风险命中 |
| 外部风险命中 | 外部库返回风险 |
| 已复核 | 审核人员已处理该风险 |

### 6.2 弹窗提示

审核人员点击订单或审核通过时，如果存在未处理黑名单命中，弹窗提示：

```text
疑似黑名单用户，请严格审核

命中来源：内部黑名单 / 外部黑名单
命中规则：身份证匹配 / 手机号匹配 / 姓名手机号匹配
风险类型：逾期 / 拒不归还 / 恶意退款等
历史处理：关联订单、处理结论、加黑时间

[查看风险详情] [继续审核] [转主管复核] [审核不通过]
```

继续审核必须填写处理意见。审核通过时要保存“已知悉风险并继续”的操作日志。

## 7. 从订单加黑

订单列表和订单详情可提供「加黑名单」入口。

| 场景 | 规则 |
|---|---|
| 审核不通过后加黑 | 需要填写风险类型和原因 |
| 租后逾期加黑 | 可从租后详情发起，关联催收证据 |
| 拒不归还加黑 | 需关联归还/设备证据 |
| 客诉恶意行为加黑 | 需关联客诉记录 |
| 解除黑名单 | 需要主管权限和解除原因 |

加黑不是关闭订单的替代动作。订单状态仍按订单流程处理。

---

## 8. 外部黑名单库预留

后期外部黑名单库对接建议抽象为独立接口服务。

| 能力 | 说明 |
|---|---|
| 查询接口 | 传入脱敏/加密后的身份要素或后端安全查询 |
| 返回字段 | 命中状态、风险等级、风险类型、来源、建议 |
| 回调/异步 | 如果外部查询慢，可异步回写订单风险 |
| 配置 | 开关、超时时间、重试次数、费用承担方 |
| 日志 | 查询请求、结果、错误码、耗时 |

外部黑名单库不可直接把完整敏感数据写入普通日志。

## 9. 批量导入

| 功能 | 说明 |
|---|---|
| 导入模板 | 姓名、手机号、身份证号、风险类型、原因、关联订单 |
| 导入校验 | 身份字段至少一个，重复数据合并提示 |
| 导入结果 | 成功数、失败数、重复数 |
| 导入日志 | 文件名、操作人、时间、结果摘要 |
| 权限 | 仅风控主管或管理员 |

导入文件应进入受控附件存储，不能被普通用户下载。

## 10. 权限与日志

| 动作 | 权限 | 日志 |
|---|---|---|
| 查看列表 | 风控/审核 | 脱敏查看 |
| 查看完整身份 | 高权限 | 敏感查看日志 |
| 新增加黑 | 风控/主管 | 加黑原因、来源 |
| 从订单加黑 | 审核/租后/主管 | 关联订单和证据 |
| 编辑风险等级 | 主管 | 变更前后值 |
| 解除黑名单 | 主管/管理员 | 解除原因 |
| 导入 | 管理员 | 文件和结果摘要 |
| 外部查询 | 系统/审核 | 请求编号和结果摘要 |
| 命中后继续审核 | 审核客服 | 已知悉风险和审核意见 |
