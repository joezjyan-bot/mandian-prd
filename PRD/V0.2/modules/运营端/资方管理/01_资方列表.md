# 资方列表

> **来源**: P1-2
> **核心**: V1 不开放资方端,运营客服代操作所有资方动作

## 1. 页面定位

运营端 · 资方管理 · 资方列表。运营客服查看所有资方,管理资方账号、偏好、钱包、合同模板。

## 2. 顶部指标

- 资方总数
- 启用资方数 / 暂停资方数 / 禁用资方数
- 资方钱包总余额
- 本月新增资方数
- 资方累计放款总额(在租中订单设备款合计)

## 3. 筛选

- 资方名称 / 联系人 / 联系方式
- 状态(active / paused / disabled)
- 可放款品类(手机 / 电动车 / 不限)
- 单笔限额范围
- 总额限额范围
- 当前在租累计范围
- 钱包余额范围(可用、不足两档)
- 创建时间范围

## 4. 列表字段

- 资方 ID
- 资方主体名称
- 联系人 / 联系方式
- 可放款品类(标签)
- 单笔限额
- 总额限额
- 当前在租累计
- 剩余可放款额度 = 总额限额 - 当前在租累计
- 钱包余额
- 待扣费金额(资方钱包待扣费队列)
- 在租订单数
- 历史合作订单数
- 状态
- 创建时间 / 创建人
- 操作:详情 / 编辑 / 启用-暂停-禁用 / 钱包流水 / 合同模板

## 5. 列表交互

- 默认按"剩余可放款额度"从高到低排序(便于分配时优先看)
- 钱包余额不足 / 待扣费 / 暂停状态高亮显示

## 6. 资方详情页

进入详情后展示:
- 基础信息(主体名称、联系人、合作合同号、创建信息)
- 偏好配置(品类、单笔限额、总额限额、其他备注)
- 钱包余额 + 历史流水
- 在租订单列表 + 历史订单
- 合同模板配置(默认 / 专属)
- 操作日志

## 7. 状态控制

- active → paused:暂停接新单(不影响在租订单)
- active → disabled:禁用(不可分配新单 + 在租订单继续)
- paused/disabled → active:恢复

操作需二次确认 + 备注

## 8. 权限

- 运营管理员 + 财务主管 + 资方管理客服:可见
- 资方主体本身:V1 不开放访问

## 9. 操作日志

任何资方创建/修改/状态变更:留日志
