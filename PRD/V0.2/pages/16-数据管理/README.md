# 数据管理

## 模块入口

- 左侧菜单：`数据管理`
- 子菜单：
  - `获客明细表`：`/dataManagement/index`
  - `导出数据下载`：`/dataManagement/dataDownloadCenter`

## 获客明细表

### 入口与路由

- 菜单路径：`数据管理 / 获客明细表`
- 路由：`/dataManagement/index`
- 面包屑：`首页 / 数据管理 / 获客明细表`
- 页面标题：`获客明细表`
- 单位提示：`单位: 笔`

### UI 结构

```text
获客明细表
├─ 维度选择区
│  ├─ 时间区间
│  ├─ 营销场景
│  ├─ 商户名称
│  ├─ 商品ID
│  └─ 渠道来源
├─ 锁定开关
├─ 条件添加区
│  └─ +
└─ 图表区
   ├─ 左侧条件/维度块
   └─ 右侧图表画布
```

### 维度选择

| 字段 | 类型 | 说明 |
|---|---|---|
| 时间区间 | 复选框 | 当前默认勾选且不可取消 |
| 营销场景 | 复选框 | 可作为统计维度 |
| 商户名称 | 复选框 | 可作为统计维度 |
| 商品ID | 复选框 | 可作为统计维度 |
| 渠道来源 | 复选框 | 可作为统计维度 |

### 点击路径

```text
点击 +
└─ 未锁定时 Toast：请先锁定查询条件

点击 锁定
└─ 开关变为 解锁，维度复选框禁用

再点击 +
└─ 弹窗：输入条件
   ├─ * 时间区间：开始时间 ~ 结束时间
   ├─ 取消
   └─ 确定
```

### 状态与反馈

- 未锁定条件时不能新增查询条件。
- 锁定后可新增条件。
- 弹窗取消后返回图表页。

### 待确认问题

- `营销场景`、`商户名称`、`商品ID`、`渠道来源` 被勾选后，`输入条件` 弹窗是否动态增加对应字段，需继续复测。
- 图表类型、导出入口、刷新频率、数据口径需要产品补充。

## 导出数据下载

### 入口与路由

- 菜单路径：`数据管理 / 导出数据下载`
- 路由：`/dataManagement/dataDownloadCenter`
- 面包屑：`首页 / 数据管理 / 导出数据下载`

### UI 结构

```text
导出数据下载
├─ 操作区
│  └─ 刷新
└─ 表格
   ├─ 文件名称
   ├─ 操作人
   ├─ 导出时间
   ├─ 状态
   └─ 操作
```

### 按钮与状态

| 点击位置 | 结果 | 风险 |
|---|---|---|
| `刷新` | 刷新导出任务列表 | 低 |
| 表格 `操作` | 当前为空数据未见具体动作，预期为下载/查看失败原因 | 中，下载可能包含敏感数据 |

### 空状态

- 当前为空数据，展示 `暂无数据`。

### 权限与审计

- 该页面集中承载各模块导出任务，必须记录导出来源、筛选条件、操作人、文件有效期。
- 下载动作应做独立权限控制。
