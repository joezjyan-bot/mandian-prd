# 商家中心：服务中心

## 菜单结构

```text
服务中心
└─ 常见问题
```

## 页面：常见问题

- 路由：`/serviceCenter`
- 详情页路由：`/serviceCenter/problemDetail/{id}`

## Tab 分类

```text
常见问题
├─ 常见问题
├─ 产品问题
├─ 售后问题
├─ 发货问题
├─ 订单问题
├─ 支付问题
├─ 审核问题
└─ 隐私问题
```

## 表格字段

| 字段 | 说明 |
|---|---|
| ID | 问题编号 |
| 标题 | 问题标题 |
| 创建时间 | 内容创建时间 |
| 操作 | `查看详情` |

## 点击反馈

| 操作 | 点击结果 |
|---|---|
| 切换 Tab | 刷新该分类问题列表 |
| 查看详情 | 跳转问题详情页 |
| 浏览器返回/面包屑 | 返回常见问题列表 |

## 详情页

```text
问题详情
├─ 面包屑：首页 / 服务中心 / 问题详情
├─ 问题标题
└─ 答案正文
```

## 重构建议

1. 服务中心内容由运营端配置，商家端只读。
2. 新系统可增加全文搜索、分类筛选、是否解决反馈。
3. 详情正文需支持富文本但限制脚本，避免 XSS。
