# 异常与客服联系

> **来源**: P1-5

## 1. 异常类型

C 端可能遇到的异常:
- 实名 / 人脸识别失败
- 信用评估不通过
- 支付失败
- 合同签署失败
- 设备问题
- 账单争议
- 客户身体/财务变故无法继续

## 2. 联系门店

订单详情 · 通用区:
- 显示门店名称 + 地址 + 联系电话
- 一键拨号(支持小程序拨号 API)
- 一键查地图(导航)

## 3. 联系平台客服

订单详情 · 通用区 + 客户中心:
- 在线客服(IM 接入)
- 客服电话(显示号码 + 一键拨号)
- 工单提交

### 3.1 在线客服 IM
- 支付宝小程序:支付宝官方客服 SDK
- 微信小程序:微信客服 SDK
- H5:平台自建 IM 或第三方(Udesk等)

工作时间:9:00-21:00(可配)
非工作时间:可留言,客服上班后回复

### 3.2 工单提交
客户可提交工单:
- 工单类型(下拉)
- 描述
- 上传附件(图片/PDF)
- 关联订单(如适用)

工单系统:
- 工单 ID
- 状态(待处理 / 处理中 / 已解决 / 已关闭)
- 跟进记录
- 客户可查询工单进度

## 4. 申诉

如客户对订单状态/费用有异议:
- 在订单详情点"申诉"
- 填申诉原因 + 期望结果
- 提交后客服核实

## 5. 紧急联系

如客户遇到紧急情况(身份证被冒用 / 账户被盗等):
- 显眼位置展示"紧急客服"通道
- 24 小时人工接入(或留言)
- 客服立即响应

## 6. 客户中心

C 端"我的"页面入口:
- 我的订单
- 实名信息(查看 / 更新)
- 收货地址管理
- 客服中心(在线 / 电话 / 工单)
- 帮助文档
- 隐私政策 / 用户协议

## 7. 数据模型

- 工单表 `customer_ticket`
- 申诉表 `customer_appeal`
- IM 会话表(若自建)
