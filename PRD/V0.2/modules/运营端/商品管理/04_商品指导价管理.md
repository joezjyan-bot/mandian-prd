# 商品指导价管理

> **V0.2 Peer Review 修订(2026-05-27)v1.0**:
> - 本文落实同行小程序能力吸收任务 8。
> - 指导价仅指**设备价区间**,不是月租价、套餐价或 C 端展示价。
> - 数据库字段、API 路径、英文枚举保持底层兼容;本文只定义页面与业务规则。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 商品指导价管理 |
| 所属端 | 运营端 |
| 入口路径 | 商品管理 > 指导价管理 |
| 使用角色 | 平台超管、运营主管、商品运营、审核客服 |
| 核心目标 | 给标准商品 + 规格组合维护设备价合理区间,在商家上架、商品审核、办单助手和客服改套餐时提供校验依据 |

---

## 2. 核心口径

1. **指导价 = 设备价区间**,用于校验办单助手中的“设备价”字段。
2. 指导价不等于每期应付、不等于首期应付、不等于 C 端商品展示价。
3. C 端不展示“指导价”字样;客户只看订单总额、首期应付、每期应付、保证金、逾期费用等已确认费用。
4. 指导价按“标准商品 + 规格组合 + 生效日期”维护。
5. 商家订单超出指导价上限可提交,但必须弹窗警告并记录日志;联营订单、平台订单超出上限直接拦截。
6. 低于指导价下限不拦截,但进入商品审核和办单助手日志,便于运营复核异常低价。
7. 指导价变更不影响历史订单和已生成订单快照。

---

## 3. 数据来源

| 字段 | 说明 |
|---|---|
| 标准商品 | 平台标准商品库中的品牌、型号、类目 |
| 规格组合 | 由商品规格矩阵生成,如“全新 + 黑色 + 有电池” |
| 指导价下限 | 该设备规格的最低合理设备价 |
| 指导价上限 | 该设备规格的最高合理设备价 |
| 生效日期 | 起始生效日 |
| 失效日期 | 可为空;为空表示当前有效 |
| 维护人 | 最近修改人 |
| 修改原因 | 必填,用于审计 |

## 5. 办单助手展示

办单助手在“设备价”字段附近展示指导价区间:

```text
设备价
¥ [________]
指导价区间:¥2,000 - ¥3,500

提示:
- 商家订单超出上限可继续提交,但需确认自担价格异常风险。
- 联营订单 / 平台订单超出上限不可提交。
```

**注意**:这里的“设备价”是内部计算和合同变量字段,不是客户侧营销展示价。

## 6. 校验规则

| 合作模式 | 设备价低于下限 | 设备价在区间内 | 设备价高于上限 |
|---|---|---|---|
| 商家订单 | 允许,记录日志 | 允许 | 弹窗警告 + 二次确认后允许 |
| 联营订单 | 允许,记录日志 | 允许 | 直接拦截 |
| 平台订单 | 允许,记录日志 | 允许 | 直接拦截 |

商家订单高于上限的弹窗:

```text
设备价已超出平台指导价上限。

当前设备价:¥4,200
指导价区间:¥2,000 - ¥3,500

该订单为商家订单,您可以继续提交,但由此产生的客户投诉、审核风险、履约风险由商家自行承担。

[取消修改] [我已知晓,继续提交]
```

## 7. 与其他模块联动

| 模块 | 联动内容 |
|---|---|
| 商品规格矩阵 | 每个 SKU 读取对应指导价区间 |
| 商家商品管理 | 商家填写设备价时显示指导价;超上限按合作模式处理 |
| 商品审核 | 审核客服看到设备价与指导价差异 |
| 办单助手 | 设备价输入框实时校验 |
| 订单审核 | 客服改套餐或改设备价时复用同一校验 |
| 操作日志 | 记录超上限提交、低于下限提交、指导价变更 |

## 8. 权限

| 角色 | 查看 | 新增/修改 | 批量导入 | 删除/失效 |
|---|---|---|---|---|
| 平台超管 | 是 | 是 | 是 | 是 |
| 运营主管 | 是 | 是 | 是 | 是 |
| 商品运营 | 是 | 提交修改申请 | 否 | 否 |
| 审核客服 | 是 | 否 | 否 | 否 |
| 商家角色 | 仅在商品/办单助手页面查看区间 | 否 | 否 | 否 |

删除不物理删除,只允许设置失效日期。

## 10. 验收要点

1. 商家订单设备价超指导价上限只警告不拦截。
2. 联营订单 / 平台订单设备价超指导价上限必须拦截。
3. 办单助手、商家商品、运营商品审核、订单改套餐使用同一套指导价规则。
4. C 端不展示指导价、合作模式、资金来源、资方等内部字段。
