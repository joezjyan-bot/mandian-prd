## V0.2 (2026-05-23) - 重构基线整理中

### 新增（Added）

- 新增 `PRD/V0.2/01_重构决策基线.md`，明确租赁系统与中控台边界、三类订单统一口径、办单助手三入口、商品规格、门店入驻、员工账号、保留/删除/后置模块。
- 新增 `PRD/V0.2/02_三端模块地图.md`，按运营端、商家 PC 端、门店手机端、C 端、渠道端、资方端拆分模块和数据互通关系。
- 新增 `PRD/V0.2/03_商品办单助手订单核心链路.md`，先固定商品、规格、增值服务、商品同步、三入口办单助手、价格方案、订单生成、审核归属、分账和失败态。
- 新增 `PRD/V0.2/04_UI风格与页面原型规范.md`，确认后续 UI 默认参考 `finance-zhongtai/static/admin-v2` 企业财务中台风格，并规定页面级 PRD 必须包含完整 UI 图、字段说明、点击路径和失败态。

### 修订方向（Changed）

- `低费率订单` 后续统一改名为 `平台订单`。
- 商品审核升级为完整的商品管理，不只保留审核权限。
- `全新/二手` 从商品单选改为同一商品下的规格。
- 门店端删除配资充值概念，普通商家暂不作为出资方。
- 第三方中控台调整为租赁系统外部的轻量能力管理系统，只管 token、密钥、额度、计费、套餐、调用记录；具体厂商业务接口代码保留在租赁系统内。
