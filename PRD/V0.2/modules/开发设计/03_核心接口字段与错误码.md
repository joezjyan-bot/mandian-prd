# 核心接口字段与错误码

> 所属：开发设计
> 目标：在核心接口清单基础上补请求字段、响应字段、错误码和事件日志要求。最终接口协议可继续细化为 OpenAPI。

---

## 1. 通用请求头

| 字段 | 必填 | 说明 |
|---|---|---|
| `Authorization` | 是 | 登录态或访问令牌 |
| `X-Request-Id` | 是 | 请求追踪编号 |
| `X-Idempotency-Key` | 写接口必填 | 幂等键，防重复提交 |
| `X-Client-Type` | 是 | ops / merchant_pc / store_mobile / customer / channel |
| `X-App-Version` | 否 | 客户端版本 |
| `X-Device-Id` | 否 | 登录设备 |

---

## 2. 通用响应结构

```json
{
  "code": "OK",
  "message": "success",
  "request_id": "req_xxx",
  "data": {},
  "warnings": []
}
```

分页响应：

```json
{
  "items": [],
  "page": 1,
  "page_size": 20,
  "total": 0
}
```

---

## 3. 通用错误码

| 错误码 | 说明 | 处理 |
|---|---|---|
| `AUTH_REQUIRED` | 未登录 | 跳转登录 |
| `AUTH_EXPIRED` | 登录过期 | 刷新或重新登录 |
| `PERMISSION_DENIED` | 无权限 | 隐藏按钮或提示无权限 |
| `VALIDATION_FAILED` | 参数校验失败 | 返回字段级错误 |
| `IDEMPOTENCY_CONFLICT` | 幂等键冲突 | 返回首次提交结果或提示重复 |
| `RESOURCE_NOT_FOUND` | 资源不存在 | 提示已删除或无权限 |
| `STATUS_CONFLICT` | 状态不允许当前动作 | 刷新页面 |
| `BUSINESS_RULE_BLOCKED` | 业务规则阻断 | 展示阻断原因 |
| `CHANNEL_UNAVAILABLE` | 第三方通道不可用 | 提示稍后重试或切通道 |
| `CALLBACK_VERIFY_FAILED` | 回调验签失败 | 进入回调异常 |
| `AMOUNT_NOT_BALANCED` | 金额不平 | 进入财务异常 |
| `SENSITIVE_OPERATION_CONFIRM_REQUIRED` | 高风险动作需二次确认 | 弹出确认或二次验证 |

## 4. 商家入驻接口字段

### 4.1 创建入驻申请

请求字段：

| 字段 | 说明 |
|---|---|
| `mobile` | 注册手机号 |
| `password` | 登录密码，加密传输 |
| `sms_code` | 短信验证码 |
| `invite_code` | 渠道推广码，可空 |
| `agree_terms` | 是否同意协议 |

响应字段：

| 字段 | 说明 |
|---|---|
| `application_id` | 入驻申请编号 |
| `merchant_id` | 商家草稿编号 |
| `store_id` | 门店草稿编号 |
| `next_step` | 下一步资料类型 |

事件日志：

- `MERCHANT_APPLICATION_CREATED`
- `CHANNEL_ATTRIBUTION_BOUND`

### 4.2 上传资质附件

请求字段：

| 字段 | 说明 |
|---|---|
| `application_id` | 入驻申请编号 |
| `file_type` | business_license / id_card_front / id_card_back / storefront |
| `file_id` | 附件中心编号 |
| `ocr_result` | 识别结果 |

响应字段：

- `qualification_status`
- `recognized_fields`
- `missing_fields`

事件日志：

- `MERCHANT_QUALIFICATION_UPLOADED`
- `OCR_RESULT_SAVED`

### 4.3 审核入驻申请

请求字段：

| 字段 | 说明 |
|---|---|
| `application_id` | 申请编号 |
| `review_result` | approved / rejected |
| `reject_reason` | 拒绝原因 |
| `enabled_capabilities` | 开通能力：商家订单、联营订单、平台订单 |

响应字段：

- `merchant_status`
- `store_status`
- `wallet_id`
- `default_config_version`

错误码：

- `QUALIFICATION_INCOMPLETE`
- `RECEIVE_ACCOUNT_NOT_VERIFIED`
- `ESIGN_AUTH_NOT_COMPLETED`

## 5. 办单助手接口字段

### 5.1 生成报价

请求字段：

| 字段 | 说明 |
|---|---|
| `order_type` | store / dividend / platform |
| `merchant_id` | 商家 |
| `store_id` | 门店 |
| `product_id` | 商品 |
| `sku_id` | 规格 |
| `rent_mode` | long / short |
| `billing_unit` | hour / day / week / month |
| `term` | 租期数量 |
| `down_payment` | 首期实付 |
| `selected_service_ids` | 增值服务 |
| `funding_ratio` | 联营订单使用 |

响应字段：

| 字段 | 说明 |
|---|---|
| `quote_snapshot_id` | 报价快照 |
| `price_items` | 保证金、租金、服务费、公证费、留购价 |
| `bill_plan_preview` | 账单预览 |
| `funding_preview` | 联营订单出资和分账预览 |
| `config_version_id` | 使用配置版本 |
| `expires_at` | 报价过期时间 |

错误码：

- `PRODUCT_NOT_SYNCED`
- `SKU_DISABLED`
- `PRICE_RULE_MISSING`
- `FUNDING_RATIO_REQUIRED`
- `SHORT_RENT_DEVICE_REQUIRED`

### 5.2 生成下单二维码

请求字段：

- `quote_snapshot_id`
- `valid_minutes`
- `channel_id`
- `staff_id`

响应字段：

- `qrcode_id`
- `qrcode_payload`
- `qrcode_image_url`
- `expires_at`

事件日志：

- `QUOTE_CREATED`
- `QRCODE_CREATED`

## 9. 回调接口字段

所有第三方回调统一入库字段：

| 字段 | 说明 |
|---|---|
| `channel` | 通道 |
| `event_type` | 回调类型 |
| `external_event_id` | 外部事件编号 |
| `raw_headers` | 原始请求头 |
| `raw_body` | 原始报文 |
| `signature` | 签名 |
| `received_at` | 接收时间 |

处理结果：

- `verify_status`
- `process_status`
- `related_object_type`
- `related_object_id`
- `retry_count`
- `error_message`
