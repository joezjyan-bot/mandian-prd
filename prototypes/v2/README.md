## 守的红线(已在原型中落实)

1. **C 端红线**:`c-customer-miniapp.html` 完全不出现合作模式(商家/联营/平台)、资方/资金来源、分账、平台服务费拆分、商家结算价、风控字眼。客户只看到"租门店一台设备 + 每期应付 + 保证金 + 到期可还/可续/可买"。
2. **运营后台**订单详情抽屉里内置一条 C 端红线提示,提醒内部字段不可下发 C 端。
3. **门店端人脸隐私红线**:交付材料里"人机合照"标注"仅存比对结果",不暗示建人脸库。
4. **商家可见边界**:`merchant-pc.html` 标注商家不可见平台利润 / 服务费拆分 / 其他商家;财务人员可提现但不可改结算账户;店员不可编辑商品、不可见成本价。
