# 商家中心：数据管理

## 菜单结构

```text
数据管理
└─ 导出数据下载
```

## 页面：导出数据下载

- 路由：`/dataManagement/dataDownloadCenter`
- 页面定位：统一展示商家端导出任务和文件下载入口。

## UI 结构

```text
导出数据下载
├─ 刷新
└─ 表格
   ├─ 文件名称
   ├─ 导出时间
   ├─ 状态
   └─ 操作
```

## 点击反馈

| 控件 | 点击结果 | 重构要求 |
|---|---|---|
| 刷新 | 列表刷新，当前实测为空状态 | loading 和失败提示 |
| 操作/下载 | 空表未出现 | 下载需校验文件权限、有效期、脱敏策略 |

## 状态设计建议

| 状态 | 展示 |
|---|---|
| 生成中 | 禁用下载，显示进度或排队中 |
| 成功 | 可下载，显示文件大小和过期时间 |
| 失败 | 展示失败原因，允许重新生成 |
| 已过期 | 禁用下载，可重新导出 |

## 风险边界

商家端导出可能包含订单、手机号、收货地址、身份证、账单等敏感数据。新系统应统一进入导出任务中心，不允许列表页直接下载生产文件。
