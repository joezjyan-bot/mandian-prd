# 门店办单手册模板

> 适用对象：门店老板 / 店员

---

## 1. 操作入口

```text
门店手机端
  -> 办单助手
  -> 选择商家订单 / 联营订单 / 平台订单
```

---

## 2. 三入口说明

| 入口 | 说明 | 审核主体 |
|---|---|---|
| 商家订单 | 门店自有订单，自己审核 | 商家/门店 |
| 联营订单 | 门店出设备，平台资方补资金 | 运营端 |
| 平台订单 | 门店递单给平台，平台主控 | 运营端 |

---

## 3. 正常流程

```text
选择订单类型
  -> 选择商品
  -> 选择规格
  -> 选择租期和方案
  -> 选择增值服务
  -> 联营订单填写出资比例
  -> 系统计算报价
  -> 生成二维码
  -> 客户扫码下单
```

---

## 4. 页面截图占位

1. 门店首页，标注办单助手入口。
2. 三入口选择页，标注三个订单类型。
3. 商品选择页，标注商品、规格、全新/二手。
4. 价格计算页，标注首期、后期、保证金、服务费、公证费、留购价。
5. 联营订单比例选择页，标注 20%-80%。
6. 二维码页，标注客户扫码入口和有效期。

---

## 5. 注意事项

- 店员账号只允许办单和处理待办，不允许查看钱包。
- 商品不在办单助手展示时，检查是否已同步。
- 当前长租办单不选择具体设备；发货/交付时填写设备识别码。
- 联营订单和平台订单提交后由运营端审核。
- 客服可在后台改价，但会生成改价日志和价格快照。

---

## 6. 数据联动

办单助手生成二维码后：

- 生成报价快照。
- 记录商品、规格、租期、金额、增值服务。
- 客户扫码后生成订单。
- 订单同步到运营端、商家端、门店端。
- 联营订单同步出资比例到内部资金来源分配和分账。
