# 账龄分析与 Vintage 报表(对外融资报表)

> **所属端**:运营端 > 财务管理 > 报表中心
> **版本**:v1.0 / 2026-05-29
> **性质**:**对外财务报表**(融资尽调用)。银行/资方在融资时要求提供应收账龄与世代质量分析。本表按"可导出、可盖章、口径可追溯、可下钻到订单"的对外报表标准设计,不是内部风控看板。
> **口径原则**:本报表所有口径必须**唯一写死**(见 全局/13 可配置项规范 §2——口径不可配);仅"账龄分桶边界、Vintage 时间粒度"等展示参数可配。涉及银行认可口径处标【需财务/银行确认】,由财务与对接银行确认后定稿,开发先按推荐口径留结构。

---

## 0. 为什么要这张表

1. 你们通过应收账款转让 / 赊销(蓝海银行等)融资,银行需评估这批应收的回款质量。
2. 银行做资产尽调时通常要两类表,常常都要:
   - **账龄分布表(Aging Schedule)**:某时点全部在租应收按逾期天数分桶,看"此刻的健康度"。
   - **Vintage 世代分析(Vintage Analysis)**:按起租/放款月份分世代,追踪各世代随账龄推移的累计逾期率,看"趋势与坏账预测"。
3. 两张表配套:分布表看静态、Vintage 看动态。只给一张银行通常会再要另一张。

## 1. 关键口径定义(唯一写死,涉及银行处标确认)

### 1.1 应收口径(分子/分母基础)

| 口径项 | 推荐定义 | 说明 |
|---|---|---|
| 在租应收余额 | 客户名下"已生成账单计划、尚未结清"的应付租金本金合计 | 【需财务/银行确认】是否含留购价、是否含逾期费用 |
| 应收是否含留购价 | 推荐:**不含**留购价(留购为客户可选行为,非确定应收) | 银行口径可能要求含,标确认 |
| 应收是否含逾期费用 | 推荐:本金与逾期费用分列,主表只算本金 | 逾期费用单列附注 |
| 计入范围 | 推荐:仅长租安心用订单(有分期应收);短租体验租不计入 | 短租无分期应收,口径不同 |

### 1.2 逾期口径(与逾期管理/状态字典对齐)

| 口径项 | 推荐定义 |
|---|---|
| 逾期起算 | 账单应付日次日起算逾期天数(与逾期管理一致) |
| 逾期天数归桶 | 按"订单级最早未付逾期期"的逾期天数归桶(银行通常要订单级,不要拆期级) |
| 一笔订单只进一个桶 | 是。订单按其最严重(最长逾期)的未付期归入对应账龄桶 |
| 门店风险逾期率分母 | 该门店所有已进入履约(`IN_FULFILLMENT`)及之后的订单,含已完成(`NORMAL_SETTLED`)、已逾期、已坏账,不含未起租订单 |

> 【需财务/银行确认】部分银行要求"期级"(每期分别归桶)。本表默认订单级,期级作为可选导出维度预留。

说明:门店风险逾期率是内部风控/运营看板口径,用于店铺详情和审核页只读参考;账龄分布与 Vintage 是对外财务报表口径。二者可同源,但不得互相替代表头、分母或导出口径。

### 1.3 Vintage 世代口径

| 口径项 | 推荐定义 |
|---|---|
| 世代划分时间 | 推荐:**放款日**(资方放款/应收转让生效日)所在月份 | 融资场景银行多按放款口径 |
| 备选 | 起租日(合同生效日)所在月份 | 【需财务/银行确认】二选一定死 |
| 账龄轴(MOB) | Month on Book:放款后第 1 个月、第 2 个月…… | 标准世代轴 |
| 世代逾期指标 | 推荐:各世代到各 MOB 时的"累计逾期率"=该世代累计出现过逾期(达 X 天)的订单本金 ÷ 该世代起始本金 | X 天阈值【需确认】,常用 30+ 或 90+ |

## 2. 报表一:账龄分布表(Aging Schedule)

### 2.1 表结构(某时点快照)

| 账龄桶 | 订单数 | 应收本金余额 | 占比 | 客户数 |
|---|---|---|---|---|
| 未逾期(当前) | … | … | …% | … |
| 逾期 1-30 天 | … | … | …% | … |
| 逾期 31-60 天 | … | … | …% | … |
| 逾期 61-90 天 | … | … | …% | … |
| 逾期 90 天以上 | … | … | …% | … |
| **合计** | … | … | 100% | … |

- 账龄桶边界(1-30/31-60/…)做成**可配**(展示参数),但默认采用上表标准桶(银行常用)。
- 支持按维度筛选:全平台 / 单门店 / 单资金来源(平台自有 vs 蓝海赊销)/ 单城市 / 单费率方案。
- 每个桶可下钻到订单明细列表(订单号、客户脱敏、本金、逾期天数、门店)。
- 表头标注:统计时点(精确到日)、口径版本、数据生成时间。

### 2.2 附注(银行常要)

- 逾期费用单独一行附注(不并入本金)。
- 准备金/拨备(如有计提规则)单列。
- 与总账(财务 08)应收科目余额勾稽,差异说明。

## 3. 报表二:Vintage 世代分析(Vintage Analysis)

### 3.1 表结构(三角矩阵)

行 = 世代(放款月份);列 = 账龄 MOB(放款后第 N 月);格 = 该世代到该 MOB 的累计逾期率。

| 世代\MOB | M1 | M2 | M3 | M4 | M5 | M6 | … | 世代起始本金 | 订单数 |
|---|---|---|---|---|---|---|---|---|---|
| 2026-01 | 0.2% | 0.5% | 0.9% | 1.3% | 1.6% | 1.8% | … | ¥X | … |
| 2026-02 | 0.1% | 0.4% | 0.8% | 1.1% | … | | | ¥X | … |
| 2026-03 | 0.3% | 0.7% | … | | | | | ¥X | … |
| …(近月世代账龄短,右侧空) | | | | | | | | | |

- 呈下三角:越早的世代账龄越长(列越多),近月世代只有前几个 MOB。
- 指标默认"累计逾期率(90+ 或 30+,阈值可配但口径写死)";可切换看"当期逾期率""坏账率"。
- 可按 资金来源 / 门店 / 费率方案 拆分出多张 Vintage 表(银行常要按资产包看)。
- 颜色梯度辅助(越红越坏),但导出 Excel 保留纯数值版。

### 3.2 解读价值(供运营/融资说明)

- 同一 MOB 列纵向比:近月世代是否比早期世代更快变坏(承保质量下降信号)。
- 同一世代横向看:逾期率爬升斜率,预测最终坏账。
- 这是银行判断"这批应收能不能放款、给什么折扣率"的核心依据。

## 4. 导出与合规(对外报表必须)

1. 支持导出 Excel(银行要可编辑核验)与 PDF(盖章版)。
2. 导出文件含:口径说明页、统计时点、数据生成时间、口径版本号、制表人。
3. 每个汇总数可下钻到订单级支持银行抽样核验。
4. 导出操作进审计日志(谁、何时、导了哪个口径/时点的表);对外报表导出为高危操作,限财务角色。
5. 报表数据与总账(财务 08)、资金穿透(财务 07)勾稽一致;不一致须有差异说明,不得直接出表。
6. 历史报表快照留存:每次对外出具的报表存档(时点+口径+数值),便于银行复查与版本追溯。

## 5. 字段与数据来源

```text
报表底层(只读聚合,不新增资金流水):
- 数据源:账单计划(应收)、账单实付、逾期状态(逾期管理)、放款/转让记录(财务07)、订单主表(起租日/放款日/门店/资金来源/费率方案)
- ar_aging_snapshot      -- 账龄分布表快照(时点+桶+本金+订单数+口径版本)
- vintage_cohort         -- 世代定义(cohort_month、起始本金、订单数、cohort_basis=放款日/起租日)
- vintage_metric         -- 世代×MOB×指标值(累计逾期率/坏账率,含阈值口径)
```

- 报表为**聚合只读**,不产生资金流水,不参与五边对账(但数值须与对账后的应收勾稽)。
- 计算频率可配(推荐每日生成账龄快照、每月刷新 Vintage)。

## 6. 待确认清单(交财务/对接银行)

| 项 | 推荐 | 待确认 |
|---|---|---|
| 应收是否含留购价 | 不含 | 银行口径 |
| 应收是否含逾期费用 | 不含(单列) | 银行口径 |
| 逾期归桶 订单级/期级 | 订单级 | 银行口径 |
| 世代划分 放款日/起租日 | 放款日 | 银行口径 |
| Vintage 逾期阈值 | 30+ 或 90+ | 银行口径 |
| 账龄桶边界 | 1-30/31-60/61-90/90+ | 可按银行模板调 |
| 是否按资金来源/资产包分表 | 是(平台自有 vs 蓝海赊销分开) | 确认资产包划分 |

> 开发先按推荐口径建结构,以上确认后写入口径配置(口径定义写死,仅桶边界/阈值/粒度作为参数)。

## 7. 衔接

- 数据来源:逾期管理、财务 06(流水)、财务 07(放款/转让/资金穿透)、财务 08(总账应收勾稽)、订单主表。
- 与门店静默率/逾期率(补充需求三)区别:那是**内部风控看板**(实时参考、驱动停单);本表是**对外财务报表**(融资尽调、口径严格、可盖章导出)。两者数据可同源,但用途、口径严格度、导出要求不同,不可混为一张表。
- 配置遵循 全局/13:口径写死,桶边界/阈值/频率可配。
