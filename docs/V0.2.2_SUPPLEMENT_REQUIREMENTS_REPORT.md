## 三、仍需配置或联调固化项

| 项目 | 当前处理 | 是否阻塞 PRD 开发契约 |
|---|---|---|
| 人设备合照 AI 核验阈值 | 高置信通过 / 低置信人工复核 / 不通过拦截的分数线由运营与算法/供应商联调后配置 | 不阻塞;字段和三档流程已固定 |
| 客服派单并发上限 | 按队列 / 角色可配置,具体默认值由运营配置 | 不阻塞;接口与状态机已固定 |
| 连续未接自动小休阈值 | 作为可配置项预留 | 不阻塞 |
| 门店静默率阈值、计算频率、告警阈值 | 按监管锁心跳能力和运营策略配置 | 不阻塞 |
| 逾期率分母和低首付定义 | 交付前最终复核 D-1 / D-2 已要求并入主文档定稿,以对应主文档为准 | 不阻塞 |

## 四、验收口径

1. S1 上传人设备合照后必须能产生 AI 辅助核验结果、评分、供应商流水号和人工最终结论;不得因 AI 结果绕过人工最终确认。
2. S2 客服必须先上班才进入派单池;会话派给某客服后 3 分钟内只允许该客服接单或主管强制改派;转派、接单、超时均留操作日志。
3. S3 店铺详情和订单审核视图必须展示门店静默率、逾期率只读参考;停单和关低首付均为 V1 人工操作并写审计日志。
4. 三项需求均不得改动数据库字段名、API 路径或英文枚举值;如需新增实现字段,以 OpenAPI / ERD / 页面契约主文档为准。
