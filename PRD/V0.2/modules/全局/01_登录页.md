# 全局登录页

> 页面级 PRD 草案。  
> 当前设计以本页为准;外部参考材料已从当前仓库清理。

---

## 1. 页面说明

| 项 | 内容 |
|---|---|
| 页面名称 | 登录页 |
| 所属端 | 运营端 / 商家 PC 端 / 中控台 |
| 入口路径 | 访问后台域名未登录时自动跳转 |
| 使用角色 | 平台运营、审核客服、财务、商家老板、商家员工、中控台管理员 |
| 核心目标 | 让不同角色通过扫码、验证码、密码登录进入对应后台 |

---

## 3. 字段说明

| 字段 | 类型 | 必填 | 说明 | 校验/失败提示 |
|---|---|---|---|---|
| 登录主体 | 文本 | 是 | 根据访问域名显示系统名 | 域名未配置时显示默认系统名 |
| 登录方式 | 标签 | 是 | 验证码登录 / 密码登录 / 扫码登录 | 无权限的登录方式隐藏 |
| 手机号 | 文本 | 验证码登录必填 | 用于接收验证码 | 手机号格式错误 |
| 用户名 | 文本 | 密码登录必填 | 可为账号名或手机号 | 账号不存在或已禁用 |
| 密码 | 密码 | 密码登录必填 | 登录密码 | 密码错误 |
| 验证码 | 文本 | 验证码登录必填 | 短信验证码 | 验证码错误或过期 |
| 二维码 | 图片 | 扫码登录必填 | 支付宝/微信扫码授权 | 二维码过期，请刷新 |

---

## 4. 按钮与交互

| 按钮/控件 | 点击后 | 成功结果 | 失败结果 |
|---|---|---|---|
| 验证码登录 | 切换表单 | 显示手机号、验证码、获取验证码 | 无 |
| 密码登录 | 切换表单 | 显示用户名、密码 | 无 |
| 获取验证码 | 调用短信接口 | 倒计时开始 | 提示发送失败原因 |
| 登录 | 提交登录 | 进入对应后台首页 | 显示账号/权限/验证码错误 |
| 刷新二维码 | 重新拉取二维码 | 二维码更新 | 提示网络异常 |

---

## 5. 权限与跳转

| 角色 | 登录后跳转 | 说明 |
|---|---|---|
| 平台管理员 | 运营端工作台 | 可见全量配置与财务 |
| 审核客服 | 订单待审核列表 | 不可操作财务核心配置 |
| 财务 | 财务工作台 | 可见结算、打款、对账 |
| 商家老板 | 商家工作台 | 可管理商品、订单、员工、钱包 |
| 商家员工 | 办单助手 / 待办任务 | 只允许办单和查看待办，不展示钱包和配置 |
| 中控台管理员 | 中控台工作台 | 管理客户、Token、套餐、调用统计 |

---

## 6. 登录保持

| 项 | 规则 |
|---|---|
| PC 后台 | 默认 7 天免登录，可由系统配置调整 |
| 门店 H5 | 同一手机登录后默认 30 天免登录 |
| 员工账号 | 同一设备默认 7 天免登录，老板可强制下线 |
| 高风险操作 | 修改财务配置、打款、提现审核、查看完整敏感信息时需要二次验证 |

> 免登录时长为产品默认建议，最终可在系统安全配置中调整。

---

## 7. 异常状态

| 场景 | 页面提示 | 后续动作 |
|---|---|---|
| 密码错误 | 账号或密码错误 | 可重试 |
| 验证码错误 | 验证码错误或已过期 | 重新获取 |
| 账号禁用 | 账号已禁用，请联系管理员 | 不允许登录 |
| 无权限 | 当前账号无此系统权限 | 返回登录页 |
| 二维码过期 | 二维码已过期 | 刷新二维码 |
| 网络异常 | 网络异常，请稍后重试 | 保留表单内容 |

---

## 8. 数据与日志

1. 记录登录成功、登录失败、验证码发送、扫码授权、退出登录。
2. 记录设备信息、IP、端类型、登录方式。
3. 敏感字段不进入前端日志。
4. 连续失败需要触发风控限制，例如图形验证码、短时锁定、管理员提醒。
